    **Theorem 1: Smoothness of rho_Lambda(x) (FISH-01)**

    Prove that rho_Lambda(x) is a smooth function of lattice position x for systems with exponentially decaying correlations.

    Structure of the proof:

    1. State the setup: Consider a lattice Hamiltonian H on Z^d (or a finite lattice with OBC) with finite-range interactions. Let |psi_0> be the unique ground state. Let Lambda(x) = {x, x+1, ..., x+|Lambda|-1} be a subsystem of size |Lambda| centered at position x. Define rho_Lambda(x) = Tr_{Lambda^c}(|psi_0><psi_0|).

    2. State the Hastings-Koma exponential clustering theorem (CMP 265, 781, 2006):
       If H has a spectral gap gamma > 0 above the ground state, then for any local observables A supported on region X and B supported on region Y:
       |<psi_0|AB|psi_0> - <psi_0|A|psi_0><psi_0|B|psi_0>| <= C ||A|| ||B|| min(|X|, |Y|) exp(-d(X,Y)/xi)
       where xi = O(v_LR / gamma) and v_LR is the Lieb-Robinson velocity.

    3. Apply to the subsystem shift: rho_Lambda(x+1) - rho_Lambda(x) is a trace-class operator whose norm is controlled by the correlations between the boundary sites of Lambda(x) and Lambda(x+1). Specifically, Lambda(x+1) differs from Lambda(x) by removing site x and adding site x+|Lambda|. The operator norm of the difference is bounded by:
       ||rho_Lambda(x+1) - rho_Lambda(x)||_1 <= C' exp(-|Lambda|/xi)
       because the site being removed (x) and the site being added (x+|Lambda|) are separated by distance |Lambda|.

    4. For higher derivatives: The k-th discrete derivative involves subsystems separated by k sites, giving:
       ||Delta^k rho_Lambda(x)||_1 <= C_k exp(-(|Lambda| - k)/xi)
       which is finite for all k < |Lambda|, establishing C^infinity smoothness in the discrete sense (all discrete derivatives bounded).

    5. State the result as a theorem with explicit conditions:
       - Conditions: finite-range Hamiltonian, unique ground state, spectral gap gamma > 0
       - Conclusion: rho_Lambda(x) is smooth in x with all derivatives bounded by exp(-|Lambda|/xi)
       - Constants: xi = O(v_LR/gamma), C depends on interaction strength and subsystem geometry

    6. Verify for SWAP Hamiltonian: The SWAP Hamiltonian H = J sum_<ij> P_ij on OBC has nearest-neighbor interactions (finite-range). For finite N, the ground state is unique (no accidental degeneracy for generic J). The spectral gap is positive for finite N (ED verifiable). Therefore Theorem 1 applies.

    7. Note the gapless caveat: In the thermodynamic limit for d>=2, the Heisenberg AFM (n=2) is GAPLESS with Neel order. The exponential clustering fails, replaced by algebraic decay. Smoothness in the algebraic regime is deferred to Phase 33 (CORR-03). State this honestly.

    **Theorem 2: Positive-definiteness of g_ij(x) (FISH-02)**

    Prove that the SLD Fisher metric is strictly positive at interior lattice points.

    Structure of the proof:

    1. State: The SLD Fisher metric g(x) = sum_{m,n: p_m+p_n>0} 2|<m|d_x rho|n>|^2 / (p_m + p_n) is manifestly non-negative (all terms are non-negative).

    2. Prove g(x) > 0: It suffices to show d_x(rho) != 0. On OBC, rho_Lambda(x) varies with x (no translation symmetry). At interior points, rho_Lambda(x) != rho_Lambda(x+1) because the subsystem sees different environments (distance to left boundary != distance to right boundary). Therefore d_x(rho) = (rho(x+1) - rho(x-1))/2 != 0, and g(x) > 0.

    3. Quantitative lower bound: From the exponential clustering bound, ||d_x rho||_1 >= delta(x) > 0 where delta(x) depends on the asymmetry of the boundary effects. For a point at distance d_L from the left boundary and d_R from the right boundary:
       delta(x) ~ |exp(-d_L/xi) - exp(-d_R/xi)| > 0 when d_L != d_R.

    4. Handle the midpoint: At x = N/2 (equidistant from both boundaries), d_L = d_R and the leading asymmetry vanishes. But subleading terms (from the actual ground state structure, not just exponential decay) generically produce d_x(rho) != 0. Verify numerically from Plan 01 data.

    5. Handle rank-deficient case: If rho_Lambda(x) is rank-deficient at some point, the SLD metric can diverge (Safranek 2017). In this case, use the Bures metric as a continuous fallback: g_Bures(x) = 2(1-sqrt(F(rho(x), rho(x+1)))). This is always well-defined and agrees with g_SLD/4 at full-rank points.

    6. Cross-check against Plan 01 data: Verify that g(x) > 0 at all interior points for all N and |Lambda| values. Report the minimum g value and where it occurs.

    **Theorem 3: Distance recovery (FISH-03)**

    Prove that d_Fisher(x,y) / d_lattice(x,y) -> const > 0 for |x-y| >> xi.

    Structure of the proof:

    1. Bulk translation invariance: In the interior of a large OBC lattice (both x and y far from boundaries), the Fisher metric g(x) is approximately constant. Define g_bulk = lim_{N->inf} g(N/2) for the midpoint metric. The corrections decay exponentially from the boundaries:
       g(x) = g_bulk + O(exp(-x/xi)) + O(exp(-(N-x)/xi))

    2. From bulk constancy to distance recovery: The geodesic distance in 1D is:
       d_Fisher(x,y) = integral_x^y sqrt(g(z)) dz = sqrt(g_bulk) * |x-y| * (1 + O(corrections))
       where the corrections come from boundary effects and are exponentially small when both x, y are in the bulk.

    3. Therefore: d_Fisher(x,y) / d_lattice(x,y) -> sqrt(g_bulk) as N -> infinity for bulk points.
       The constant sqrt(g_bulk) is positive because g_bulk > 0 (from Theorem 2).
