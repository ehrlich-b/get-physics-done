  <name>Task 1: Implement QFIM module and validate on TFI benchmark</name>
  <files>code/fisher_metric.py, tests/test_fisher_metric.py</files>
  <action>
    Build the QFIM computation module on top of the existing ed_entanglement.py infrastructure.

    **1. Core QFIM functions (code/fisher_metric.py):**

    Include ASSERT_CONVENTION line:
    % ASSERT_CONVENTION: units=natural, lattice_spacing=1, fisher_metric=SLD, normalization=g_F_equals_4_times_g_Bures, eigenvalue_threshold=1e-14, boundary=OBC

    Functions to implement:

    a) `reduced_states_sweep(psi, N, subsystem_size, bc='obc')`:
       - For each lattice position x = 0, 1, ..., N-subsystem_size, compute rho_Lambda(x) = partial_trace(|psi><psi|, sites={x, x+1, ..., x+subsystem_size-1}).
       - Use existing `partial_trace` from ed_entanglement.py.
       - Return list of (x, rho_x) pairs.
       - Verify: Tr(rho_x) = 1 and rho_x >= 0 (Hermitian, positive semidefinite) at each x.

    b) `compute_qfim(rho_list, threshold=1e-14)`:
       - Input: list of (x, rho_x) from reduced_states_sweep.
       - For each interior point x (not first or last), compute d_x(rho) via central finite difference: d_x(rho) = (rho_{x+1} - rho_{x-1}) / 2.
       - Eigendecompose rho_x = sum_m p_m |m><m| using np.linalg.eigh.
       - Compute SLD Fisher metric scalar (1D): g(x) = sum_{m,n: p_m+p_n > threshold} 2|<m|d_x(rho)|n>|^2 / (p_m + p_n).
       - Return array of (x, g_x, eigenvalues_of_rho_x, rank_of_rho_x).
       - Handle rank-deficient case: if p_m + p_n < threshold, skip that (m,n) pair. Record which points have reduced rank.

    c) `compute_bures_metric(rho_list)`:
       - For each interior point x, compute Bures metric via fidelity:
         F = [Tr(sqrt(sqrt(rho_x) * rho_{x+1} * sqrt(rho_x)))]^2
         g_Bures(x) = 2(1 - sqrt(F)) / (delta_x)^2 where delta_x = 1 (lattice spacing).
       - Use eigendecomposition-based matrix sqrt: sqrt(A) = U * diag(sqrt(eigenvalues)) * U^dag.
       - Return array of (x, g_Bures_x).

    d) `fisher_geodesic_distance(g_values, x1, x2)`:
       - In 1D, geodesic distance = sum of sqrt(g(x)) * a over the path from x1 to x2.
       - d_Fisher(x1, x2) = sum_{x=x1}^{x2-1} sqrt(g(x)) * 1 (lattice spacing).
       - Return geodesic distance.

    e) `compute_distance_ratios(g_values, positions)`:
       - For all pairs of interior bulk positions, compute d_Fisher/d_lattice.
       - Return array of ratios and statistics (mean, std, min, max).

    **2. TFI benchmark (tests/test_fisher_metric.py):**

    a) `test_tfi_benchmark()`:
       - Construct TFI Hamiltonian for N=12, OBC, at h/J = 0.5, 1.0, 1.5 using existing construct_tfi_1d.
       - Compute ground state, reduced states, Fisher metric at each h.
       - Verify: g(h=1.0) > g(h=0.5) and g(h=1.0) > g(h=1.5) (critical enhancement at QPT).
       - This validates the QFIM algorithm against the known Zanardi et al. 2007 result.

    b) `test_sld_bures_agreement()`:
       - For N=8, TFI at h/J=0.5, compute SLD and 4*Bures at all interior points.
       - Assert relative error < 1e-10 at all full-rank points.
       - This is the critical cross-validation identity: g_F^{SLD} = 4 * g_Bures (Braunstein-Caves 1994, Safranek 2017).

    c) `test_pbc_gives_zero()`:
       - Compute Fisher metric for Heisenberg N=8 PBC.
       - Assert max|g| < 1e-12 (translation invariance kills the metric).

    d) `test_metric_symmetry_and_positivity()`:
       - For N=8 OBC SWAP, verify g(x) > 0 at all interior points.
       - (In 1D, g is a scalar, so symmetry is trivial. Positivity is the key check.)

    e) `test_trace_preservation()`:
       - Verify Tr(rho_x) = 1 +/- 1e-14 and all eigenvalues of rho_x >= 0 at every position.

    **Anti-patterns to avoid:**
    - Do NOT use scipy.linalg.logm (fails for rank-deficient states).
    - Do NOT use PBC for any physics result (gives g=0).
    - Compute the REDUCED state Fisher metric (partial trace first), not the full-state Fubini-Study metric.
    - For the SLD denominator, threshold on p_m + p_n (not individual eigenvalues).
  </action>

    3. PBC sanity check: max|g| < 1e-12 on PBC (translation symmetry)

    5. Positivity: all eigenvalues of rho_x >= 0 at every x
