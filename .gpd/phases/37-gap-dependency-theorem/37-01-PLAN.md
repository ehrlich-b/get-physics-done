    **Step 4: KMS => entropy stationarity via Gibbs variational principle.**

    ROUTE A (Gibbs variational principle): The Gibbs variational principle (Pusz-Woronowicz CMP 58, 1978; Araki CMP 1975): among all states with the same modular energy <K_A>, the KMS state has the highest entropy. Equivalently: the free energy F = <K_A> - S is minimized by the KMS state. At the KMS state, first-order stationarity: delta F = 0, hence delta <K_A> = delta S at first order. Combined with the entanglement first law (delta S = delta <K_A>, exact): delta S_total = 0 at first order.

    ROUTE B (Direct relative entropy, more streamlined): For any state omega' near the KMS state omega, the relative entropy S(omega' || omega) >= 0 with equality iff omega' = omega. Expanding: S(omega' || omega) = <K_A>_{omega'} - S(omega') - (free energy of omega) = delta <K_A> - delta S + O(delta^2). At first order: delta <K_A> - delta S >= 0 with equality for the reference state. But the entanglement first law gives delta S = delta <K_A> exactly, so the bound is saturated. This means the reference (KMS) state is at a stationarity point: delta(S - <K_A>) = 0.

    Present BOTH routes. Route B is more direct and avoids explicitly invoking the Gibbs variational principle name, though the mathematical content is the same (positivity of relative entropy). Assumptions used: the KMS state exists (from Steps 1-3); the relative entropy / free energy framework applies. For type III algebras (thermodynamic limit), this is rigorous (Araki). For type I algebras (finite lattice), the Gibbs state e^{-K}/Z trivially satisfies this.
