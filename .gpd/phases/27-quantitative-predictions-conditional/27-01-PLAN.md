    **Section 1: Minimum Initial Entropy from the Landauer Bound**

    Derive the minimum entropy deficit required to sustain self-modeling:

    1. From Phase 25: W_cycle >= T * I(B;M) per self-modeling update cycle (k_B = 1).
    2. The maximum extractable work from a system with entropy S < S_max = ln(d): W_available = T * (S_max - S) = T * Delta_S, where Delta_S = S_max - S is the entropy deficit.
    3. The number of self-modeling cycles before exhaustion: N_exhaust = W_available / W_cycle = Delta_S / I(B;M). This is the maximum number of update cycles the system can sustain before equilibrating.
    4. For self-modeling to be sustained: N_exhaust >= 1, i.e., Delta_S >= I(B;M), i.e., S <= S_max - I(B;M). This gives S_min_deficit = I(B;M): the system must start with at least I(B;M) nats below maximum entropy.
    5. In terms of absolute entropy: S_initial <= S_max - I(B;M). The minimum initial entropy for N cycles of self-modeling: S_initial <= S_max - N * I(B;M).

    DIMENSIONAL CHECK at this point: [S] = [nats] = dimensionless. [I] = dimensionless. [S_max] = dimensionless. [N] = dimensionless integer. Consistent.

    **Section 2: Cosmological Estimate**

    Apply the Landauer bound to the observable universe (with explicit model-dependence flags):

    1. S_max for the observable universe: Bekenstein bound on the cosmological horizon gives S_max ~ 10^{122} (in nats, order of magnitude). This is a COSMOLOGICAL INPUT, not derived from the framework. [MODEL-DEPENDENT: depends on cosmological model, horizon size.]
    2. Penrose's estimate: S_initial ~ 10^{88} at the Big Bang (from the Weyl curvature hypothesis and black hole entropy arguments). [EXTERNAL ESTIMATE: derived from GR + black hole thermodynamics, not from this framework.]
    3. The Landauer constraint: S_initial <= S_max - N * I(B;M). For macroscopic self-modeling: I(B;M) ~ ln(d_M) where d_M is the effective dimension of the model subsystem. For a human-scale self-modeler with ~10^{10} neurons at ~10 bits each: I ~ 10^{10} * ln(2) ~ 7 * 10^9 nats. [MODEL-DEPENDENT: depends on what counts as "self-modeling" and the model complexity.]
    4. Number of update cycles in the age of the universe: If the update rate is ~ 10 Hz (neural timescale) and the universe is ~4*10^{17} s old, then N ~ 4 * 10^{18} cycles. [MODEL-DEPENDENT: depends on update rate and timescale.]
    5. Total entropy deficit needed: N * I ~ 4 * 10^{18} * 7 * 10^9 ~ 3 * 10^{28} nats. This is VASTLY smaller than Penrose's deficit S_max - S_initial ~ 10^{122} - 10^{88} ~ 10^{122}.

    STATE THE KEY RESULT: The Landauer bound on self-modeling gives a constraint on S_initial that is many orders of magnitude WEAKER than Penrose's estimate. The Landauer bound requires only S_initial <= S_max - 10^{28}, while Penrose requires S_initial ~ 10^{88} (a deficit of ~10^{122}). The Landauer bound is NECESSARY but not SUFFICIENT to explain the observed initial entropy. The framework says "the Past Hypothesis is needed for self-modeling" but does not predict HOW MUCH initial entropy there was.

    SELF-CRITIQUE CHECKPOINT: Is this result trivial? Partially -- it shows the Landauer bound is a very weak constraint on initial entropy. But it is NOT trivial because: (a) it proves the QUALITATIVE requirement (S_initial < S_max), (b) it provides a lower bound on the entropy deficit, and (c) the weakness of the bound is itself informative (tells us the Landauer bound is not the limiting factor; gravitational entropy dominates).

    **Section 3: Exhaustion Timescale**

    Compute the timescale over which a self-modeler exhausts its free energy reservoir:

    1. From Phase 23: E^N(rho_B) = (1-p)^N rho_B + (1-(1-p)^N) I/2, where p = sin^2(Jt).
    2. The body entropy increases as S_N = S(E^N(rho_B)). After N steps, the entropy approaches S_max = ln(d_B).
    3. The mutual information I(B;M) decreases as the body approaches the maximally mixed state. When rho_B -> I/d_B, the body carries no information, and I(B;M) -> 0.
    4. The timescale for equilibration: N_eq ~ 1/p = 1/sin^2(Jt). For weak coupling (Jt << 1): N_eq ~ 1/(Jt)^2. Each step takes time t, so tau_eq ~ t/(Jt)^2 = 1/(J^2 t).
    5. The exhaustion timescale for self-modeling: tau_exhaust = N_exhaust * t_cycle where t_cycle is the update period and N_exhaust = Delta_S / I(B;M).

    **Section 4: Experiential Density Profile rho(t)**

    Evaluate the experiential density as a function of cosmological time:

    1. The experiential density: rho(I) = I * (1 - I/H) where H = S(rho_B) is the body entropy (following the project notation; note: H here is the entropy, not the Hamiltonian -- clarify notation).

    NOTATION CLARIFICATION: In the roadmap and Paper 5, rho = I(B;M)(1 - I/H) uses H for entropy of the body. In the derivation files, this is S(rho_B). To avoid confusion with the Hamiltonian H = JF, write this as: rho(I) = I * (1 - I/S_B) where S_B = S(rho_B).

    2. This is a parabola in I with zeros at I = 0 and I = S_B, peak at I = S_B/2 with rho_max = S_B^2/4 (verified in derivations/25-landauer-self-modeling.md, Section 4).
    3. As the universe evolves from low-entropy past (small S_B) toward equilibrium (S_B -> S_max):
       - Initially: S_B is small, so rho_max = S_B^2/4 is small. Self-modeling has limited "room."
       - At intermediate times: S_B is moderate, and I can be substantial. rho can be large.
       - Late times: S_B -> S_max but I -> 0 (equilibration from Phase 23). rho -> 0.
    4. The key question: at which cosmological epoch does rho peak? This depends on the COMPETITION between increasing S_B (more entropy headroom) and decreasing I (equilibration destroys correlations).
    5. Two limiting regimes:
       (a) I tracks the maximum: I = S_B/2 at all times. Then rho = S_B^2/4, and rho increases monotonically with S_B. But this is unrealistic -- maintaining I = S_B/2 requires active self-modeling that itself costs free energy.
       (b) I decreases via equilibration: I(N) ~ I_0 * (1-p)^N (from Phase 23 dynamics applied to mutual information). Then rho = I_0(1-p)^N * (1 - I_0(1-p)^N / S_B). This peaks at some intermediate N and then decays to zero.
    6. STATE HONESTLY: The rho profile over cosmological time is MODEL-DEPENDENT because:
       (a) The mapping from update steps N to cosmological time t requires specifying J and t_step (model-dependent).
       (b) The body entropy S_B(t) depends on the cosmological model.
       (c) Whether the self-modeler actively maintains I or lets it decay depends on the self-modeling protocol.
    7. What CAN be said model-independently:
       (a) rho = 0 at thermal equilibrium (proved in Phase 25).
       (b) rho peaks at I = S_B/2 for fixed S_B.
       (c) rho -> 0 as the universe approaches heat death.
       (d) The existence of self-modelers NOW implies we are NOT at equilibrium (entropy gradient theorem, Phase 26).
