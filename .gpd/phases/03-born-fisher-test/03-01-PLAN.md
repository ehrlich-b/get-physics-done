<objective>
Implement quantum information utilities and run the static diagonal-state test (Test A) of the Born-Fisher-Experiential conjecture. This plan builds the core computational infrastructure (density matrices, partial traces, von Neumann entropy, quantum MI) and then uses it to sweep the (p, alpha) parameter space for diagonal qubit/qutrit states, extracting the half-saturation curve alpha_half(p) and comparing Born-rule vs non-Born distributions.

Purpose: Determine whether the static half-saturation condition I_vN = S_vN/2 reveals any special property of Born-rule probability assignments. A clear positive, negative, or "uninformative" verdict is the deliverable.

Output: quantum_info.py (reusable utilities), test_a_static.py (sweep + analysis), test_a_results.json (numerical data), heatmap and alpha_half figures.

Note on scipy: scipy is not installed. All root-finding (for alpha_half) must be implemented via manual bisection using numpy only. This is straightforward for 1D root-finding on a monotone function.
</objective>

  <name>Task 1: Implement quantum information utilities and validate against known benchmarks</name>
  <files>simulations/born_fisher/quantum_info.py</files>
  <action>
    Create a self-contained Python module with numpy-only implementations of core quantum information functions. All computations in nats (natural log). Design for arbitrary Hilbert space dimension d (not hardcoded to d=2).

    Functions to implement:

    1. `von_neumann_entropy(rho, eps=1e-15)`: Compute S_vN = -Tr(rho ln rho) via eigendecomposition.
       - Use numpy.linalg.eigh (Hermitian).
       - Filter: set eigenvalues < eps to 0. Convention: 0*ln(0) = 0.
       - Return scalar (nats).

    2. `partial_trace(rho_bm, d_b, d_m, trace_over='M')`: Partial trace of bipartite density matrix.
       - rho_bm is (d_b*d_m) x (d_b*d_m) matrix.
       - Reshape to (d_b, d_m, d_b, d_m), then trace over the specified subsystem axis.
       - trace_over='M' returns d_b x d_b matrix; trace_over='B' returns d_m x d_m matrix.

    3. `quantum_mutual_information(rho_bm, d_b, d_m)`: Compute I_vN(B;M) = S_vN(B) + S_vN(M) - S_vN(BM).
       - Uses partial_trace and von_neumann_entropy.

    4. `experiential_density(rho_bm, d_b, d_m)`: Compute rho_Q = I * (1 - I/S_B).
       - Handle S_B = 0 edge case: return 0.0 (convention: 0/0 = 0).
       - Return tuple (rho_Q, I_vn, S_B, ratio) for diagnostics.

    5. `diagonal_bm_state(p, alpha, d=2)`: Construct diagonal (p, alpha)-parametrized BM state.
       - For d=2: rho_BM = p * |0><0| (x) rho_M^(0) + (1-p) * |1><1| (x) rho_M^(1)
         where rho_M^(0) = diag(alpha, 1-alpha), rho_M^(1) = diag(1-alpha, alpha).
       - For d=3: Generalize with d pointer states and tracking matrix alpha_ij where diagonal = alpha, off-diagonal = (1-alpha)/(d-1).
       - Return (d*d) x (d*d) density matrix.

    6. `bisect_root(f, a, b, tol=1e-12, max_iter=100)`: Simple bisection root-finder.
       - f(a) and f(b) must have opposite signs. Return root to tolerance tol.
       - This replaces scipy.optimize.brentq.

    7. `validate_density_matrix(rho, label="", tol=1e-12)`: Assert Tr=1, Hermitian, PSD.
       - Print warnings for violations. Return bool.

    After implementing all functions, run the following validation suite WITHIN the module (as a `if __name__ == '__main__'` block):

    Sanity checks (must ALL pass):
    (a) Tr and PSD: diagonal_bm_state(0.5, 0.75, d=2) passes validate_density_matrix
    (b) Pure Bell state: rho = |Phi+><Phi+| where |Phi+> = (|00>+|11>)/sqrt(2). Verify I_vN = 2*ln(2) to 1e-12.
    (c) Product state: rho_B (x) rho_M with rho_B = diag(0.7, 0.3), rho_M = diag(0.4, 0.6). Verify I_vN = 0 to 1e-12.
    (d) S_vN(I/2) = ln(2) = 0.6931471805599453 to 1e-12.
    (e) S_vN(I/3) = ln(3) = 1.0986122886681098 to 1e-12.
    (f) Araki-Lieb: for 10 random diagonal states, verify 0 <= I_vN <= 2*min(S_B, S_M).
    (g) Symmetry: for diagonal states, I_vN(p, alpha) = I_vN(1-p, alpha) (swap p <-> 1-p).
    (h) Eigenvalue cutoff stability: S_vN for a near-pure state at eps={1e-15, 1e-14, 1e-13} agrees to < 1e-13.
    (i) Perfect tracking: alpha=1 gives I_vN/S_vN = 1.0 for all p != 0,1.
    (j) No tracking: alpha=0.5 (for d=2) gives I_vN = 0.

    Print PASS/FAIL for each check with the numeric values.
  </action>
  <verify>

    6. validate_density_matrix catches invalid states (test with non-PSD matrix).
