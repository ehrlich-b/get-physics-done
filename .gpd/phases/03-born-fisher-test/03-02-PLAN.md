  <action>
    Create a Lindblad master equation integrator using numpy only (no scipy). Import quantum_info.py utilities from Plan 01.

    Functions to implement:

    1. `sigma_x, sigma_y, sigma_z, identity_2`: Standard Pauli matrices and 2x2 identity as numpy arrays.

    2. `build_lindblad_superoperator(H_int, collapse_ops, d_total)`: Construct the Lindblad superoperator L as a d^2 x d^2 matrix acting on vectorized rho.
       - Vectorize rho (d x d matrix -> d^2 vector) using column-stacking convention: vec(rho) = rho.flatten('F') (Fortran order).
       - Hamiltonian part: -i(H (x) I - I (x) H^T) acting on vec(rho).
       - Dissipator for each L_k: L_k (x) L_k* - (1/2)(L_k^dag L_k (x) I + I (x) L_k^T L_k*).
       - Return the full superoperator matrix.

    3. `rk4_step(L_super, rho_vec, dt)`: Single RK4 step.
       - k1 = L_super @ rho_vec
       - k2 = L_super @ (rho_vec + dt/2 * k1)
       - k3 = L_super @ (rho_vec + dt/2 * k2)
       - k4 = L_super @ (rho_vec + dt * k3)
       - return rho_vec + dt/6 * (k1 + 2*k2 + 2*k3 + k4)

    4. `integrate_lindblad(rho0, H_int, collapse_ops, t_span, n_steps, d_b, d_m)`:
       - rho0: initial d_total x d_total density matrix.
       - Build superoperator once. Integrate using RK4 with n_steps.
       - At each step: unvectorize rho, compute S_vN(B), I_vN(B;M), rho_Q using quantum_info functions.
       - Monitor: Tr(rho) should stay 1 to 1e-10; eigenvalues should be >= -1e-10.
       - Return: t_array, rho_Q_array, I_array, S_B_array, ratio_array, trace_array.

    5. `build_qubit_system(gamma_D, g)`: Build H_int and collapse_ops for the standard qubit-qubit system.
       - H_int = g * (sigma_x (x) sigma_x + sigma_y (x) sigma_y) -- exchange coupling.
       - collapse_ops = [sqrt(gamma_D) * (sigma_z (x) I_2)] -- dephasing on B subsystem.
       - Return H_int, collapse_ops.

    6. `build_asymmetric_system(gamma_0, gamma_1, g)`: Asymmetric dephasing.
       - H_int = g * (sigma_x (x) sigma_x + sigma_y (x) sigma_y).
       - collapse_ops = [sqrt(gamma_0) * (|0><0| (x) I_2), sqrt(gamma_1) * (|1><1| (x) I_2)].
       - Return H_int, collapse_ops.

    7. `initial_entangled_state(theta, d=2)`: Construct pure state |psi> = cos(theta)|00> + sin(theta)|11>.
       - Return rho = |psi><psi| as d^2 x d^2 density matrix.

    VALIDATION (in __main__ block):

    (a) Trace preservation: integrate one trajectory (theta=pi/4, gamma_D=1.0, g=1.0, T=10, n_steps=2000). Verify max |Tr(rho(t)) - 1| < 1e-10.

    (b) Positivity: verify all eigenvalues >= -1e-10 at every stored time step.

    (c) Pure dephasing: set g=0, gamma_D=1.0. The off-diagonal elements of rho_B should decay as exp(-2*gamma_D*t) = exp(-2t). Check at t=1: off-diag ~ exp(-2) = 0.135 to 5%. At t=5: off-diag < 1e-4.

    (d) Initial condition: at t=0 for theta=pi/4, verify I_vN = 2*S_B = 2*ln(2) to 1e-12 (pure-state identity).

    (e) Late-time consistency: at t=T_final (t >> tau_D), extract the diagonal state and compare I_vN/S_vN with the static calculation from Plan 01 (same p and effective alpha). Agreement to 1e-4.

    (f) Convergence: run same trajectory with n_steps=2000 and n_steps=4000. mu_Q (integral of rho_Q over time) should agree to 1e-6.

    Print PASS/FAIL for each validation check.
  </action>
