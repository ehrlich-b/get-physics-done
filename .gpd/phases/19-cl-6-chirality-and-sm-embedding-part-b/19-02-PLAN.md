    Build the explicit 32x32 matrix representation and verify all algebraic relations computationally. This covers COMP-01.

    **Implementation plan:**

    1. Build 10 Cl(10) generators as 32x32 matrices using the standard recursive construction:
       - Start from Cl(2): sigma_1, sigma_3 (2x2 Pauli matrices; skip sigma_2 since we want real representations... actually for complex Cl we can use all three)
       - Actually, use the standard construction: for Cl(2n), use 2^n x 2^n matrices built by tensor products of Pauli matrices.
       - For Cl(10): 2^5 = 32 dimensional. Build Gamma_1, ..., Gamma_10 as tensor products of Pauli matrices:

       Standard construction for Gamma matrices in Cl(2n) with n=5:
         Gamma_1 = sigma_1 x I x I x I x I
         Gamma_2 = sigma_2 x I x I x I x I
         Gamma_3 = sigma_3 x sigma_1 x I x I x I
         Gamma_4 = sigma_3 x sigma_2 x I x I x I
         Gamma_5 = sigma_3 x sigma_3 x sigma_1 x I x I
         ...and so on, following the standard recursive pattern.

       The key property: {Gamma_A, Gamma_B} = 2 delta_AB I_32.

    2. Verify ALL 55 anticommutation relations {Gamma_A, Gamma_B} = 2 delta_AB I_32 for A,B = 1,...,10.
       Use numpy: `assert np.allclose(Gamma_A @ Gamma_B + Gamma_B @ Gamma_A, 2*delta_AB*np.eye(32), atol=1e-14)`.

    3. Extract the 6 Cl(6) generators: gamma_k = Gamma_k for k = 1, ..., 6 (the generators corresponding to Im(C^3) directions).
       Verify all 21 Cl(6) anticommutation relations.

    4. Compute omega_6 = gamma_1 @ gamma_2 @ gamma_3 @ gamma_4 @ gamma_5 @ gamma_6.
       Verify:
       - omega_6 @ omega_6 = -I_32 (within 1e-14)
       - P = 0.5 * (I_32 - 1j * omega_6) satisfies P @ P = P (within 1e-14)
       - np.trace(P) = 16 (within 1e-10)
       - np.linalg.matrix_rank(P.real) = 16 (since P is a projection onto a 16-dim subspace)

    5. Compute the Cl(10) chirality operator:
       Gamma_11 = Gamma_1 @ ... @ Gamma_10
       Verify Gamma_11^2 = +I or -I (depending on convention) and compare with omega_6.
       Show omega_6 != Gamma_11 (they select DIFFERENT splittings of the 32-dim space).

    6. Construct Witt creation/annihilation operators:
       a_j = 0.5 * (gamma_{2j-1} + 1j * gamma_{2j}) for j = 1, 2, 3
       a_j_dag = 0.5 * (gamma_{2j-1} - 1j * gamma_{2j}) for j = 1, 2, 3

       Verify CAR: {a_i, a_j_dag} = delta_ij I_32, {a_i, a_j} = 0, {a_i_dag, a_j_dag} = 0.
       All 27 anticommutator checks.

    7. Find the Clifford vacuum: the simultaneous kernel of a_1, a_2, a_3.
       |0> is in the intersection of ker(a_1), ker(a_2), ker(a_3).
       Verify this space is nonempty and determine its dimension.
       (It should be 4-dimensional in the 32-dim Dirac space, corresponding to the 4 states of the "other" set of 3 Witt generators from the remaining Cl(4) structure.)

    Include ASSERT_CONVENTION line at top of file.
    Structure the code as pytest-compatible test functions.
    Print key results (eigenvalues, traces, ranks) for diagnostic purposes.

         |0>                          (Clifford vacuum within P-projected subspace)
         a_1^dag |0>                  (one quantum)
         a_2^dag |0>
         a_3^dag |0>
         a_1^dag a_2^dag |0>          (two quanta)
         a_1^dag a_3^dag |0>
         a_2^dag a_3^dag |0>
         a_1^dag a_2^dag a_3^dag |0>  (three quanta)

       The 8 "antiparticle" states come from applying the same construction to the complementary projector (1-P), or equivalently from the charge conjugation.

       IMPORTANT: The 16 states should all lie in the P-projected 16-dim subspace of the 32-dim Dirac spinor. If using the Cl(10) chirality Gamma_11 instead, the split is different. Use the omega_6 projector P from Task 1.

       Actually, re-examine: the Cl(6) Witt decomposition gives 2^3 = 8 states from the Clifford vacuum. The full 16-dim S_{10}^+ needs the remaining 8 from the Cl(4) sector (the generators Gamma_7, ..., Gamma_10 that are NOT in Cl(6)). The Cl(4) has its own Witt decomposition with 2 creation/annihilation pairs, giving 2^2 = 4 states per chirality.

       The full 16 states = 8 (from Cl(6) Witt) x 2 (from Cl(4) sector) -- but this needs careful analysis. The 16-dim representation decomposes under Cl(6) x Cl(4) as:
         S_{10}^+ = S_6^+ tensor S_4^+ + S_6^- tensor S_4^-
       where S_6^+/- are the two 4-dim Weyl spinors of Spin(6) and S_4^+/- are the two 2-dim Weyl spinors of Spin(4).

       So the 16 states are: 4 x 2 + 4 x 2 = 8 + 8 = 16.

       Construct ALL 16 states explicitly.

    2. Define the Cartan generators for the SM gauge group:

       (a) SU(3) color: two Cartan generators from the spin(6) = su(4) subalgebra.
           Specifically, the SU(3) Cartan generators are:
             T_3^color = (1/2) [gamma_1, gamma_2] (proportional to number operator a_1^dag a_1)
             T_8^color = (1/(2*sqrt(3))) ([gamma_1, gamma_2] + [gamma_3, gamma_4] - 2*[gamma_5, gamma_6])
           (The exact form depends on the embedding. Follow Furey/Todorov conventions.)

       (b) SU(2)_L weak isospin: the I_3 generator from the spin(4) = su(2)_L + su(2)_R subalgebra.
           From the Cl(4) generators Gamma_7, ..., Gamma_10:
             I_3 = (1/4) [Gamma_7, Gamma_8] (or the appropriate combination)

       (c) Weak hypercharge Y: the U(1)_Y generator.
           Y = (2/3)(n_1 + n_2 + n_3) - 1 (or the appropriate combination from Todorov/Furey)
           where n_j = a_j^dag a_j is the number operator.

       The EXACT FORM of these generators must be extracted from Plan 01's derivation and the Todorov/Furey references. The eigenvalues on the 16 states give the quantum numbers.

    3. For each of the 16 states |psi_k>, compute:
       - Y_k = <psi_k | Y | psi_k>  (weak hypercharge)
       - I3_k = <psi_k | I_3 | psi_k>  (weak isospin third component)
       - color_k = (<psi_k | T_3^c | psi_k>, <psi_k | T_8^c | psi_k>)  (color charges)

       | e_R       | +1    | 0     | singlet     |
       | u_R^r     | -1/3  | 0     | ... |
       | ... (etc for all 16) |

       NOTE: The exact Y values depend on the normalization convention. The TABLE above uses the Pati-Salam embedding convention. Use the convention from Todorov/Furey and state it explicitly. The key test is that all 16 states have DISTINCT quantum number tuples and match a KNOWN SM assignment.

    5. Print the complete quantum number table and assert all matches.
