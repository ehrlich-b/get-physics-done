  <name>Task 1: Build ED framework and compute all benchmarks</name>
  <files>code/ed_entanglement.py</files>
  <action>
    Build a self-contained Python script for exact diagonalization and entanglement entropy computation.

    PREREQUISITE: scipy must be available. If `import scipy` fails, install it: `pip install scipy` (or `pip3 install scipy` or `python3 -m pip install scipy` depending on the environment). Verify import succeeds before proceeding. If installation fails, fall back to dense numpy.linalg.eigh for N <= 16 only.

    REUSE from existing code (code/self_modeling_lr_velocity.py): Pauli matrices (sigma_x, sigma_y, sigma_z), construct_swap(n) function pattern. Do NOT copy the entire file -- extract only the needed utilities.

    The script must implement:

    1. ASSERT_CONVENTION line at top:
       % ASSERT_CONVENTION: units=natural, entropy=nats, hamiltonian_sign=H_min_energy,
       %   coupling_convention=H_sum_hxy, eigenvalue_threshold=1e-14, lattice_spacing=1

    2. Sparse Hamiltonian constructors:
       - construct_heisenberg_1d(N, J, bc='pbc'): H = (J/2) sum sigma.sigma, periodic or open BC
       - construct_tfi_1d(N, J_zz, h_x, bc='obc'): H = -J_zz sum sigma_z sigma_z - h_x sum sigma_x
       - construct_self_modeling_1d(N, J, bc='pbc'): H = J sum F_{i,i+1} (SWAP). For n=2, this IS the Heisenberg model shifted by a constant: JF = (J/2)(sigma.sigma) + (J/2)I. The energy shift does not affect the ground state or entanglement.
       - construct_heisenberg_2d(Lx, Ly, J, bc='pbc'): 2D square lattice Heisenberg
       All constructors use scipy.sparse.csr_matrix and scipy.sparse.kron.

    3. Ground state finder:
       - ground_state(H, k=1): wraps scipy.sparse.linalg.eigsh(H, k=k, which='SA')
       - Returns (eigenvalue, eigenvector)
       - For degenerate ground states: k=1 returns an arbitrary state in the ground space (fine for FM; for AFM Heisenberg PBC even N, ground state is unique in S_z=0 but eigsh sees the full space)

    4. Entanglement entropy:
       - partial_trace(psi, N, sites_A): Given state vector psi of N qubits and list of sites in A, compute rho_A via tensor reshape. The sites in A need NOT be contiguous -- handle arbitrary subsets by permuting tensor indices.
       - von_neumann_entropy(rho, threshold=1e-14): Eigendecompose rho, discard eigenvalues < threshold, compute S = -sum lambda_i ln(lambda_i)
       - entanglement_entropy_sweep(psi, N, max_L=None): Compute S(L) for contiguous blocks A = {0,1,...,L-1} for L = 1 to min(N//2, max_L)

    5. Calabrese-Cardy fitting:
       - fit_cc_pbc(S_data, N): Fit S(L) = (c/3) ln[(N/pi) sin(pi L/N)] + c_1 using scipy.optimize.curve_fit or manual least squares. Return c, c_1, R^2.
       - fit_cc_obc(S_data, N): Fit S(L) = (c/6) ln[(2N/pi) sin(pi L/N)] + c_1'. Return c, c_1', R^2.

    6. Run ALL benchmarks in a main() function:

       a) TFI critical (h/J = 1, OBC):
          - N = 8, 12, 16
          - Compute E_0, S(L) for L=1..N//2
          - Fit CC (OBC formula), extract c
          - Expected: c ~ 0.5 (Ising CFT)

       b) TFI gapped (h/J = 3, OBC):
          - N = 8, 12, 16
          - Compute E_0, S(L)
          - Expected: S(L) saturates quickly (area law)

       c) AFM Heisenberg (J = +1, PBC):
          - N = 8, 12, 16 (even N only, PBC)
          - Compute E_0, S(L)
          - Fit CC (PBC formula), extract c
          - Expected: c ~ 1 (SU(2)_1 WZW), E_0/N approaching -0.443147

       d) FM Heisenberg (J = -1, PBC):
          - N = 8, 12, 16
          - Compute E_0, S(L)
          - Expected: S(L) = 0 for all L (product ground state)

    7. Internal consistency checks (run for every computed (model, N, L)):
       - S(L) == S(N-L) to 1e-12 (pure state symmetry)
       - 0 <= S(L) <= min(L, N-L) * ln(2)
       - Tr(rho_A) == 1 to 1e-14
       - rho_A hermitian: ||rho_A - rho_A^dag|| < 1e-14
       - All eigenvalues of rho_A >= -1e-14

    8. Print all results to stdout in a structured format. Also save to JSON.

    IMPORTANT: The self-modeling Hamiltonian H = JF for n=2 differs from the standard Heisenberg by a constant shift: JF = (J/2)(sigma.sigma) + (J/2)I per bond. The ground STATE is the same. The ground state ENERGY differs by N_bonds * J/2. Verify this explicitly by comparing the two constructions.
  </action>
