  <name>Task 1: Modular Hamiltonian locality check</name>
  <files>code/modular_hamiltonian_check.py</files>
  <action>
    Build on the ED framework from Plan 01 (code/ed_entanglement.py) to compute and analyze the modular Hamiltonian.

    ASSERT_CONVENTION at top:
    % ASSERT_CONVENTION: units=natural, entropy=nats, hamiltonian_sign=H_min_energy,
    %   modular_hamiltonian=K_A_minus_ln_rho_A, eigenvalue_threshold=1e-12_for_K

    Import from ed_entanglement.py: Hamiltonian constructors, ground_state, partial_trace.

    1. compute_modular_hamiltonian(rho_A, threshold=1e-12):
       - Eigendecompose rho_A: rho_A = V diag(lambda) V^dag
       - Project onto support: keep only eigenvalues > threshold
       - K_A = -V diag(ln(lambda)) V^dag (on the support)
       - Return K_A and the condition number max(lambda)/min(lambda_above_threshold)
       - Flag WARNING if condition number > 1e10

    2. boundary_distance(i, j, sites_A, sites_B, lattice_geometry):
       - For a contiguous block A = {0,...,|A|-1} in 1D:
         - Boundary sites: last site of A (|A|-1) and first site of B (|A|)
         - Distance of matrix element K_{i,j} from boundary: min over boundary sites of (|site_of_i - boundary_site| + |site_of_j - boundary_site|)
       - Return distance for each (i,j) pair in the |A| x |A| matrix

    3. locality_analysis(K_A, sites_A, geometry):
       - For each matrix element K_{i,j}: compute |K_{i,j}| and distance from boundary
       - Bin by distance: average |K| at each distance d = 0, 1, 2, ...
       - Compute ratio: |K(d)| / |K(0)| (normalized decay)
       - Compute boundary Frobenius norm: sum |K_{ij}|^2 over boundary-adjacent pairs (d=0)
       - Compute total Frobenius norm
       - Report boundary_norm / total_norm

    4. Compute K_A locality for THREE cases:

       a) TFI gapped (h/J=3, OBC, N=16, A = sites {0,1,2,3}):
          - BENCHMARK: Peschel guarantees K is local for free fermions
          - Plot |K_{ij}| vs boundary distance
          - Expect: clear exponential decay

       b) TFI critical (h/J=1, OBC, N=16, A = sites {0,1,2,3}):
          - Critical system: locality may be weaker (power-law decay expected)
          - Plot alongside gapped case for comparison

       c) AFM Heisenberg (J=+1, PBC, N=16, A = sites {0,1,2,3}):
          - INTERACTING: no analytical prediction for decay rate
          - Plot |K_{ij}| vs boundary distance
          - This is the self-modeling lattice (n=2)

    5. For each case, report:
       - K_A matrix (numerical values)
       - Decay profile: |K(d)| / |K(0)| at each distance
       - Boundary norm fraction
       - Condition number of rho_A

    KEY SUBTLETY for PBC: With PBC and A = {0,1,2,3}, the boundary consists of TWO cuts: between site 3 and site 4, and between site N-1 and site 0. The distance from boundary must account for BOTH cuts. For site i in A, distance_to_boundary = min(i, |A|-1-i) for contiguous blocks starting at 0.

    Actually for K_A matrix elements K_{ij} where i,j label basis states within A (not physical sites), we need to think carefully. K_A is a 2^|A| x 2^|A| matrix. To analyze locality, we should examine K_A in the SITE basis: write K_A = sum_{alpha} c_alpha O_alpha where O_alpha are tensor products of local operators on sites within A. Then check which terms have support near the boundary.

    SIMPLER APPROACH (standard in literature): Treat K_A as an |A|-site operator. Write it in the computational basis. For each pair of computational basis states |s> and |s'>, the matrix element <s|K_A|s'> is nonzero. Identify which sites differ between s and s' -- this tells you the support of that matrix element. Matrix elements where only boundary-adjacent sites differ should dominate.

    SIMPLEST APPROACH (for small |A|): Expand K_A in the Pauli basis for |A| qubits:
    K_A = sum_{mu} c_mu sigma_mu where sigma_mu ranges over all Pauli strings on |A| sites.
    The coefficient c_mu = Tr(K_A sigma_mu) / 2^|A|.
    Locality means: |c_mu| is large when sigma_mu acts on sites near the boundary, small when it acts deep in the interior.
    For |A| = 4: 4^4 = 256 Pauli strings. Tractable.

    USE THE PAULI EXPANSION APPROACH for the locality analysis.
