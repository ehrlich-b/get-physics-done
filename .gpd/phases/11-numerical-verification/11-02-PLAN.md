    % ASSERT_CONVENTION: units=natural, entropy=nats, hamiltonian_sign=H_min_energy,
    %   coupling_convention=H_sum_JF_SWAP, eigenvalue_threshold=1e-14, lattice_spacing=1

    Import from ed_entanglement.py: Hamiltonian constructors, ground_state, entanglement_entropy_sweep, partial_trace, von_neumann_entropy, fit_cc_pbc.

    1D COMPUTATION (AFM Heisenberg, J=+1, PBC):

    For N = 8, 12, 16, 20:
      a) Construct H = J sum F_{i,i+1} using construct_self_modeling_1d (or equivalently construct_heisenberg_1d, since they give the same ground state)
      b) Find ground state via Lanczos
      c) Compute S(L) for L = 1 to N//2
      d) Fit three models:
         - Constant: S = a (area law)
         - Linear: S = bL + d (volume law)
         - CC log: S = (c/3) ln[(N/pi) sin(pi L/N)] + c_1 (CFT prediction)
      e) Report R^2 for all three fits
      f) For N=20, also check S(L) = S(N-L) consistency

    Also compute for J = -1 (FM) at N = 8, 16 as trivial control.

    2D COMPUTATION (AFM Heisenberg, J=+1, PBC, 4x4 square lattice):

    a) Construct 2D Heisenberg Hamiltonian on 4x4 square lattice with PBC (N=16 qubits)
       - Label sites 0..15 on a 4x4 grid: site (i,j) -> index 4*i + j
       - Bonds: horizontal (i,j)-(i,j+1 mod 4) and vertical (i,j)-(i+1 mod 4, j)
       - Total bonds: 2 * 4 * 4 = 32 (PBC)

    b) Find ground state via Lanczos

    c) Enumerate all rectangular subregions A of size (a, b) where 1 <= a <= 4, 1 <= b <= 4, (a,b) != (4,4):
       - For each (a,b), there are multiple positions (corner offsets) but by translation invariance with PBC, all positions give the same S. Pick one canonical position per (a,b).
       - |volume(A)| = a * b
       - |boundary(A)| for PBC: count bonds crossing A to A^c.
         For a rectangle a x b on a 4x4 PBC lattice:
         - If a < 4: 2*b bonds cross horizontally (left and right boundary)
         - If a == 4: 0 horizontal boundary bonds
         - If b < 4: 2*a bonds cross vertically
         - If b == 4: 0 vertical boundary bonds
         - |boundary| = 2*b*(a<4) + 2*a*(b<4)
       - SPECIAL CASES: if a==4 AND b<4, boundary = 2*4 = 8 (only vertical cuts). If a<4 AND b==4, boundary = 2*4 = 8 (only horizontal). If a==4 AND b==4, this is the full system, skip.
       - Also consider non-square subregions placed at different SHAPES but same (a,b).

    d) For each subregion: compute rho_A via partial_trace (arbitrary site subsets), compute S(A)

    e) Collect data points: (|boundary(A)|, |volume(A)|, S(A)) for all subregions

    f) Regression analysis:
       - Note: among the 15 rectangular shapes (a,b) with 1<=a<=4, 1<=b<=4, (a,b)!=(4,4),
         (a,b) and (b,a) give identical S by lattice symmetry, leaving 9 independent data points.
         Also, |boundary| takes only 5 unique values (many shapes share |boundary|=8).
       - Regress S vs |boundary|: slope, intercept, R^2, Spearman rank correlation
       - Regress S vs |volume|: slope, intercept, R^2, Spearman rank correlation
       - Report BOTH "all 15 shapes" and "9 independent shapes" regressions (they should agree
         since symmetric pairs have identical (S, |boundary|, |volume|))
       - Report R^2 and Spearman rho for both; discrimination = R^2(boundary) - R^2(volume)
       - Also consider L-shaped or non-rectangular subregions to diversify |boundary| values.
         For example, an L-shaped 3-site region on a 4x4 lattice has |boundary| different from
         any same-volume rectangle. Include at least 4-6 non-rectangular subregions if feasible
         to strengthen the regression (the partial_trace function already handles arbitrary site subsets).

    CRITICAL SUBTLETY: For the 1D case, |boundary| = 2 for ALL interior cuts (L >= 1 and L <= N-1). So S vs |boundary| regression in 1D is meaningless (no variance in x). The 1D area-law test is whether S(L) saturates (constant model) or grows. The R^2 > 0.9 for |boundary| fit and R^2 < 0.5 for |volume| fit contract criteria apply to the 2D case.

    For the 1D case, report:
    - S(L) saturation test (constant fit R^2 and linear fit R^2)
    - CC log fit (c extraction)
    - Honest statement: AFM Heisenberg 1D is gapless with c=1, so S ~ (1/3)ln(L), NOT strict area law
