<task type="auto">
  <name>Task 1: Instantiate Nachtergaele-Sims LR framework for nearest-neighbor interactions</name>
  <files>derivations/08-lr-framework.md</files>
  <action>
    Write up the Nachtergaele-Sims Lieb-Robinson bound framework, adapted to our conventions, as a self-contained reference for this phase and downstream phases.

    **Do NOT re-derive the LR bound.** The bound is a deep result (Lieb-Robinson 1972, refined by Nachtergaele-Sims 2006/2019). Invoke it as a black box and instantiate it for our specific setup.

    1. **State the LR bound (Nachtergaele-Sims form):**

       For a quantum lattice system with interaction Phi on a graph G = (V, E), local observables A supported on X and B supported on Y satisfy:

       ||[tau_t(A), B]|| <= 2||A|| ||B|| min{1, (C_a)^{-1} (e^{2||Phi||_a C_a |t|} - 1) sum_{x in X, y in Y} F_a(d(x,y))}

       where:
       - tau_t(A) = e^{iHt} A e^{-iHt} is the Heisenberg evolution
       - F_a: [0, infinity) -> (0, infinity) is the F-function (decay function)
       - C_a = sup_{x in V} sum_{y in V, y != x} F_a(d(x,y)) is the convolution constant
       - ||Phi||_a = sup_{x in V} sum_{X ni x} ||Phi(X)|| / F_a(diam(X)) is the weighted interaction norm

       Cite: Nachtergaele-Sims-Young (2019), Theorem 3.4 equivalent; see also arXiv:1011.4540 Corollary 3.5.

    2. **Define the LR velocity:**

       v_LR = 2 ||Phi||_a C_a / a

       where a is the decay parameter of the F-function, optimized to minimize v_LR.

       The LR bound implies that ||[tau_t(A), B]|| is exponentially small when d(X,Y) > v_LR |t|, establishing an effective light cone.

    3. **Choose the F-function for finite-range interactions:**

       For nearest-neighbor interactions (range 1) on any lattice, the standard choice is:

       F_a(r) = e^{-ar} for r >= 0

       This is the simplest F-function that gives a finite C_a on any lattice with bounded coordination number.

    4. **Compute C_a for specific lattices:**

       For Z^d (d-dimensional cubic lattice) with exponential F-function F_a(r) = e^{-ar}:

       C_a = sup_x sum_{y != x} e^{-a d(x,y)}

       By translation invariance, this is independent of x. For Z^1:
       C_a = 2 sum_{r=1}^{infinity} e^{-ar} = 2 e^{-a} / (1 - e^{-a}) = 2 / (e^a - 1)

       For Z^d:
       C_a = sum_{y != 0} e^{-a |y|_1} where |y|_1 is the L^1 (Manhattan) distance.

       Compute C_a explicitly for d = 1, 2, 3.

    5. **Compute ||Phi||_a for nearest-neighbor interactions:**

       For Phi({x,y}) = h_{xy} with ||h_{xy}|| = J for all edges and Phi(X) = 0 for |X| != 2:

       ||Phi||_a = sup_x sum_{y: {x,y} in E} ||h_{xy}|| / F_a(1)

       For a regular lattice with coordination number z (z = 2 for Z^1, z = 4 for Z^2, z = 6 for Z^3):

       ||Phi||_a = z J / F_a(1) = z J e^a

    6. **Compute v_LR and optimize over a:**

       v_LR(a) = 2 ||Phi||_a C_a / a = 2 (z J e^a) C_a / a

       For Z^1: v_LR(a) = 2 * (2J e^a) * (2/(e^a - 1)) / a = 8J e^a / (a(e^a - 1))

       Minimize over a > 0. For Z^1, the minimum is at a = 1, giving v_LR = 8J e / (e - 1) = 8eJ / (e - 1) approximately 12.7J.

       NOTE: The "standard" result v_LR = 2eJ for Z^1 uses a slightly different F-function convention. Reconcile carefully. The exact value depends on the precise F-function and norm convention used. What matters is: (a) the formula is correct in our conventions, (b) the benchmark matches a published value.

    7. **State the effective light cone:**

       ||[tau_t(A), B]|| is exponentially small (decaying as e^{-mu(d(X,Y) - v_LR|t|)}) outside the cone d(X,Y) = v_LR |t|. The decay rate mu depends on the chosen F-function.

    Use our conventions throughout. All energies in natural units. Lattice spacing a_lat = 1.
  </action>
  <verify>
    1. **Dimensional analysis:** [v_LR] = [J] * [C_a] / [a]. [J] = energy, [C_a] = dimensionless, [a] = 1/[length] = 1 (in lattice units). So [v_LR] = energy = 1/time (natural units). With a_lat = 1, velocity = distance/time = 1/time. Consistent.

    2. **Limiting cases:** (a) J -> 0 implies v_LR -> 0 (no interaction, no propagation). (b) a -> 0 implies C_a -> infinity and v_LR -> infinity (F-function too slow to decay; not useful). (c) a -> infinity implies ||Phi||_a -> infinity (F-function too fast; not useful). Hence optimal a is interior.

    3. **C_a convergence:** For Z^d with exponential F-function, C_a must be finite for all a > 0. Verify the series converges.

    4. **Formula consistency:** Cross-check v_LR formula against Nachtergaele-Sims (2019) and against at least one other source (e.g., arXiv:1011.4540).
  </verify>
  <done>Nachtergaele-Sims LR framework instantiated for nearest-neighbor interactions on general lattices, with explicit formulas for C_a, ||Phi||_a, and v_LR, all in our conventions.</done>

    1. **Implement compute_convolution_constant(d, a):**
       Compute C_a for Z^d lattice with exponential F-function.

       - d=2: C_a = sum_{(m,n) != (0,0)} e^{-a(|m|+|n|)} (compute numerically to convergence, cutoff at |m|+|n| = 100)
       - d=3: same structure, 3D sum

    2. **Implement compute_interaction_norm(J, z, a):**
       ||Phi||_a = z * J * e^a for nearest-neighbor with uniform coupling J and coordination number z.

    3. **Implement compute_lr_velocity(J, z, d, a):**
       v_LR = 2 * ||Phi||_a * C_a / a

    4. **Implement optimize_decay_parameter(J, z, d):**
       Minimize v_LR(a) over a > 0 using scipy.optimize.minimize_scalar (if scipy available) or golden section search.
       NOTE: scipy may not be installed. If not, implement a simple golden section search or use numpy for grid-based optimization over a = [0.1, 0.2, ..., 5.0] followed by refinement.

    5. **Heisenberg chain benchmark:**
       h_{xy} = J (sigma_x sigma_x + sigma_y sigma_y + sigma_z sigma_z) / 4
       (The 1/4 factor makes ||h_{xy}|| = J for the standard normalization.)
       Actually: the Heisenberg interaction is h = J/4 * (XX + YY + ZZ) which has eigenvalues J/4 * {-3, 1, 1, 1} so ||h|| = 3J/4.
       Be CAREFUL with normalization. Use the convention where ||h_{xy}|| = J, which means the coupling strength in the LR formula is J = ||h_{xy}||.

       For the Heisenberg chain on Z^1 (z = 2):
       - Compute v_LR with the optimal a
       - Compare to the known published value (cite source)

    6. **Trivial limit check:** Verify v_LR(J=0) = 0.

    7. **Print results:** v_LR for Z^1, Z^2, Z^3 with J = 1 (the reference energy scale).

    Use numpy for all computations. If scipy is unavailable, implement optimization manually.
