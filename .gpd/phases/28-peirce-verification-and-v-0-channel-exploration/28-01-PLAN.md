    **Octonion class (code/octonion_algebra.py):**

    1. Represent octonion as 8-component real numpy array: a = (a_0, a_1, ..., a_7) where a = a_0 + a_1*e_1 + ... + a_7*e_7.
    2. Multiplication via Fano plane with convention e_1*e_2 = e_4. The 7 Fano triples are: (1,2,4), (2,3,5), (3,4,6), (4,5,7), (5,6,1), (6,7,2), (7,1,3). For each triple (i,j,k): e_i*e_j = e_k, e_j*e_k = e_i, e_k*e_i = e_j, and reversals give minus signs.
    3. Conjugation: conj(a) = (a_0, -a_1, ..., -a_7).
    4. Norm: |a|^2 = a * conj(a) = sum(a_i^2).
    5. Provide __mul__, __add__, __sub__, __neg__, conjugate(), norm(), and real/imag parts.

    **h_3(O) element class:**

    6. Represent as (alpha, beta, gamma, x_1, x_2, x_3) where alpha,beta,gamma are real and x_1,x_2,x_3 are Octonion. This maps to the 3x3 Hermitian matrix:
       | alpha    conj(x_3)  x_2       |
       | x_3      beta       conj(x_1) |
       | conj(x_2) x_1       gamma     |

    7. Jordan product: A . B = (1/2)(AB + BA) where AB is the 3x3 octonionic matrix product. Implement the 3x3 matrix product explicitly: (AB)_{ij} = sum_k A_{ik} * B_{kj}. Each A_{ik}*B_{kj} is a single octonion product (2 factors, so Artin's theorem guarantees no associativity issues). Do NOT nest octonion multiplications.

    8. Extract the result back into (alpha', beta', gamma', x_1', x_2', x_3') format.

    **Peirce projections:**

    9. Pi_1: extract alpha component (set all others to zero).
    10. Pi_{1/2}: extract x_2, x_3 components (set alpha=beta=gamma=x_1=0).
    11. Pi_0: extract beta, gamma, x_1 components (set alpha=x_2=x_3=0).
    12. Verify Pi_1 + Pi_{1/2} + Pi_0 = Id on random elements.
