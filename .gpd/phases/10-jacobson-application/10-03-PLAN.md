    **Part E: Gap statement.**

    State the two main gaps honestly and prominently (not in a footnote):

    **Gap 1: MVEH (Assumption A5).**
    - What: the vacuum state maximizes entanglement entropy at fixed stress-energy tensor
    - Why it matters: without A5, the derivation gives delta S = delta S_UV + delta S_mat but does NOT set this to zero, so Einstein's equation does not follow
    - Physical motivation: MaxEnt reasoning (self-modeling fixed point = MaxEnt state at fixed constraints)
    - What would establish it: showing that the self-modeling equilibrium state is a maximum (not just extremum) of S at fixed T_ab in the continuum limit
    - What happens if it fails: the result becomes conditional. Paper 6 states: "IF MVEH holds, THEN Einstein's equations follow from self-modeling." This is still significant: it identifies self-modeling as a UV completion compatible with Jacobson's program, with MVEH as the single remaining bridge.
    - Note: for conformal fields, MVEH is EQUIVALENT to Einstein's equation (Jacobson 2016). This means A5 is exactly as strong as the conclusion for conformal fields. For nonconformal fields, A5 is strictly weaker than Einstein's equation (it determines only the traceless part; Lambda is free). The self-modeling lattice may produce a nonconformal IR theory, so A5 may be independently checkable.

    **Gap 2: Continuum limit.**
    - What: the self-modeling lattice produces a smooth Riemannian manifold at long wavelengths
    - Why it matters: the Raychaudhuri equation, geodesic balls, causal diamonds, and the CHM modular Hamiltonian all require a differentiable manifold
    - Physical motivation: standard Wilsonian argument (universality at long wavelengths)
    - What would establish it: a constructive proof of the continuum limit, analogous to lattice QCD -> continuum QCD. This is a hard open problem in ALL lattice approaches to quantum gravity.
    - What happens if it fails: if the continuum limit does not exist, the entire Jacobson route fails. If it exists but produces non-Riemannian geometry, the result changes (possibly f(R) gravity or torsion theories).
    - Context: this gap is shared with every lattice quantum gravity approach. It is not specific to self-modeling.

    **Gap 3 (minor): Conformal approximation.**
    - The CHM modular Hamiltonian is exact for CFT only. If the self-modeling IR theory is nonconformal, corrections O((m*R)^{2*Delta}) appear (Speranza 2016). These are suppressed at scales R << 1/m.

    **Part F: Known limits check.**

    Verify the derived Einstein equation reproduces known limits:

    1. Flat spacetime: R_abcd = 0, T_ab = 0 => G_ab = 0, Lambda g_ab = 0 consistent. Check.
    2. Linearized gravity: expand g_ab = eta_ab + h_ab with |h_ab| << 1. The linearized Einstein equation is Box h_ab = -16 pi G T_ab (in Lorenz gauge). This is the standard weak-field result. Our derivation is first-order, so it reproduces this exactly. Cross-check with LMVR 2014 (entanglement first law => linearized Einstein).
    3. Schwarzschild: vacuum (T_ab = 0) with spherical symmetry. R_ab = 0 (Ricci flat). Consistent with G_ab + Lambda g_ab = 0 when Lambda = 0.
    4. de Sitter: T_ab = 0, Lambda > 0. G_ab = -Lambda g_ab, giving constant curvature. Consistent (Jacobson-Speranza 2019).
    5. Newtonian limit: g_00 = -(1 + 2 Phi), T_00 = rho. Einstein equation gives nabla^2 Phi = 4 pi G rho (Poisson equation). G = 1/(4 eta) > 0 since eta > 0, so gravity is attractive. Correct.
