    4. Emergent speed: v_LR = 8eJ/(e-1) on Z^1 (Phase 8, Eq. 08-03.3). In the continuum, v_LR -> c (the emergent speed of light). This sets the causal structure.
    5. Newton's constant: G = 1/(4 eta) (Jacobson 2012). Using the channel capacity bound: G >= a^{d-1} / (4 log(n)). The lattice spacing a ~ l_Planck up to factors of log(n).

    **Part E: MVEH formulation.**

    State the Maximal Vacuum Entanglement Hypothesis (MVEH) precisely:

    MVEH (Jacobson 2016): Among all quantum states with the same expectation value of T_ab (the stress-energy tensor), the vacuum state maximizes the entanglement entropy S(A) for any small geodesic ball A.

    Equivalently: delta S_EE = 0 for the vacuum under first-order perturbations at fixed volume. The vacuum is an entropy extremum. Combined with the expectation that it is a maximum (not a minimum or saddle point), this gives entanglement equilibrium: delta S = 0.

    Formulate this as **Assumption A5 (Maximal Vacuum Entanglement Hypothesis)** for the self-modeling lattice:

    A5 (MVEH): The self-modeling equilibrium state, when mapped to the emergent continuum description, maximizes entanglement entropy among states with the same stress-energy tensor.

    **MaxEnt motivation for A5:**

    The MaxEnt principle (Jaynes 1957) states that, given a set of constraints, the state that maximizes entropy is the least biased. For the self-modeling lattice:
    - The self-modeling fixed point IS the state that best satisfies the self-modeling constraint
    - MaxEnt reasoning suggests this fixed point maximizes entropy subject to the self-modeling constraint
    - In the continuum limit, "subject to the self-modeling constraint" maps to "at fixed stress-energy tensor" (since T_ab encodes the energy-momentum distribution that the self-modeling dynamics maintains)

    State clearly: this MaxEnt motivation is a PHYSICAL ARGUMENT, not a proof. The gap is:
    - What self-modeling property would ESTABLISH A5? The self-modeling fixed point must be shown to be a maximum of S at fixed T_ab, not just an extremum.
    - The mapping from "self-modeling constraint" to "fixed T_ab" is part of the continuum limit and inherits its uncertainties.
