  <name>Task 1: Define and test sequential product asymmetry candidates</name>
  <files>derivations/08-dirac-candidates.md</files>
  <action>
    Define and systematically test multiple linearizations of the sequential product asymmetry as Dirac operator candidates.

    Work under the Barrett isomorphism throughout: H_sector = M_n(C), pi(a) = L_a, pi_o(b) = R_b, P = transpose.

    **Candidate A: Commutator [a, -] (simplest SWAP-odd operator)**

    For fixed a in M_n(C), define D_a^comm(X) = [a, X] = aX - Xa = (L_a - R_a)(X).

    Properties:
    - Linear in X (good -- this is a legitimate operator on M_n(C))
    - SWAP-odd: [a, X]^T = X^T a^T - a^T X^T. For a = a^T (symmetric), [a, X]^T = -[a, X^T], so it maps Sym -> Skew and Skew -> Sym. This means it anticommutes with gamma (transpose) WITHIN each sector.
    - Self-adjointness: need to check (L_a - R_a)^dagger = L_{a^dagger} - R_{a^dagger} = L_{a*} - R_{a*}. For a self-adjoint (a = a*), this equals L_a - R_a. So D_a^comm is self-adjoint on M_n(C) with the Tr inner product when a = a*.

    Test for multiple a:
    - a = diag(1,0,...,0) (rank-1 projector)
    - a = E_{12} + E_{21} (off-diagonal symmetric)
    - a = identity (gives [I, X] = 0, trivial)
    - a = diag(1,2,...,n) (distinct eigenvalues)

    For each: embed in the doubled space D = [[0, M^dag], [M, 0]] where M is the gamma-odd part acting on (H_+, H_-). Check JD = DJ using the antilinear J. Diagnose which constraint(s) are violated.

    **Candidate B: Sequential product operator sqrt(a) X sqrt(a)**

    For fixed a in M_n(C)^sa with a >= 0, define D_a^sp(X) = sqrt(a) X sqrt(a).

    Properties:
    - Linear in X
    - SWAP behavior: (sqrt(a) X sqrt(a))^T = sqrt(a)^T X^T sqrt(a)^T = sqrt(a^T) X^T sqrt(a^T). For a symmetric, this equals sqrt(a) X^T sqrt(a). So it maps Sym -> Sym and Skew -> Skew (SWAP-even). This means D_a^sp COMMUTES with gamma, NOT anticommutes.
    - Therefore: D_a^sp does NOT satisfy D gamma = -gamma D. It satisfies D gamma = +gamma D.
    - Resolution: extract the SWAP-odd part. Define D_a^{sp-odd}(X) = (1/2)(sqrt(a) X sqrt(a) - sqrt(a) X^T sqrt(a)). Or consider the "asymmetry" of the SP: sp(a, X) - sp(X, a) = sqrt(a) X sqrt(a) - sqrt(X) a sqrt(X), but this is nonlinear in X.

    Document whether any natural SWAP-odd extraction of the SP operator works.

    **Candidate C: Barrett-form motivated candidate**

    Using Barrett's D(v x X) = sum_I (K_I X + epsilon' X K_I*) x gamma_I v with epsilon' = +1 for KO-dim 6:
    - For V = C^2 with gamma acting as diag(1,-1), the SWAP-odd part requires gamma_I to include the "off-diagonal" Pauli matrix.
    - The simplest Barrett candidate: D(X_p, X_{ap}) = (K X_{ap} + X_{ap} K*, -(K X_p + X_p K*)) for some K in M_n(C).
    - This mixes particle and antiparticle sectors. Test whether K related to the sequential product structure gives a viable candidate.

    For each candidate:
    1. Verify D* = D (or identify failure)
    2. Verify D gamma + gamma D = 0 (or identify failure with Frobenius norm)
    3. Verify J_matrix conj(D) J_matrix^{-1} = D (or identify failure with Frobenius norm)
    4. If all pass: project onto moduli basis from Plan 14-01 to express as linear combination
    5. If some fail: compute projection onto moduli space, report residual

    **Identification of most natural D:**

    If no sequential product candidate works:
    - Examine the moduli space basis from Plan 14-01
    - Identify which basis elements involve left/right multiplication operators
    - Among those, identify which have the simplest self-modeling interpretation
    - Among those, identify which are related to the Barrett form with physically meaningful K_I
    - Document the most natural D with explicit rationale

    Include ASSERT_CONVENTION. Reference Plan 14-01 moduli space results.
  </action>
