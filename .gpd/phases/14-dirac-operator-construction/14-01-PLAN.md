  <name>Task 1: Derive D moduli space parameterization at general n</name>
  <files>derivations/08-dirac-moduli-space.md</files>
  <action>
    Derive the complete moduli space of self-adjoint Dirac operators D on H = C^{2n^2} satisfying D gamma = -gamma D and JD = DJ.

    Work primarily under the Barrett isomorphism (C^n tensor C^n = M_n(C) via v tensor w -> v w^T) where:
    - pi(a) = L_a (left multiplication by a)
    - pi_o(b) = R_b (right multiplication by b)
    - P = transpose: P(X) = X^T
    - gamma|_p = P (transpose on particle sector), gamma|_{ap} = -P (negative transpose on antiparticle sector)

    **Step 1: Identify gamma eigenspaces under Barrett isomorphism.**
    - Particle sector: gamma = P = transpose. Eigenvalue +1: symmetric matrices Sym_n (dim n(n+1)/2). Eigenvalue -1: skew-symmetric matrices Skew_n (dim n(n-1)/2).
    - Antiparticle sector: gamma = -P = -transpose. Eigenvalue +1: skew-symmetric matrices Skew_n (dim n(n-1)/2). Eigenvalue -1: symmetric matrices Sym_n (dim n(n+1)/2).
    - H_+ (gamma = +1) = Sym_n^p + Skew_n^{ap}, dim = n(n+1)/2 + n(n-1)/2 = n^2
    - H_- (gamma = -1) = Skew_n^p + Sym_n^{ap}, dim = n(n-1)/2 + n(n+1)/2 = n^2

    **Step 2: D gamma = -gamma D forces off-diagonal block form.**
    In the (H_+, H_-) basis: D = [[0, M^dagger], [M, 0]] where M: H_+ -> H_- is an arbitrary complex n^2 x n^2 matrix. Self-adjointness is automatic. Free data: M has n^4 complex entries = 2n^4 real parameters.

    **Step 3: Write J in the gamma-eigenspace basis.**
    Since J anticommutes with gamma (J gamma = -gamma J), J maps H_+ to H_- and H_- to H_+. Write J = [[0, J_-], [J_+, 0]] where J_+: H_+ -> H_- and J_-: H_- -> H_+.

    CRITICAL: J is ANTILINEAR. So J acts as J_matrix * complex_conjugation. When computing JD = DJ, this gives J_matrix conj(D) J_matrix^{-1} = D (NOT J_matrix D J_matrix^{-1}).

    Compute J_+ and J_- explicitly. Under the Barrett isomorphism, J(X_p, X_{ap}) = (P conj(X_{ap}), P conj(X_p)) = (conj(X_{ap})^T, conj(X_p)^T) = (overline{X_{ap}}^T, overline{X_p}^T). In the gamma eigenspace basis, track how J maps:
    - Sym_n^p (gamma +1) to some component of H_- (gamma -1)
    - Skew_n^{ap} (gamma +1) to some component of H_- (gamma -1)
    Work out the explicit sub-block structure.

    **Step 4: Impose JD = DJ as a real-linear constraint on M.**
    The condition JD = DJ with D = [[0, M^dagger], [M, 0]] and J = [[0, J_-], [J_+, 0]] (antilinear) gives:
    - J_+ conj(M^dagger) = M J_+  (from comparing (2,1) blocks, accounting for antilinearity)
    - J_- conj(M) = M^dagger J_-  (from comparing (1,2) blocks)
    These are conjugate conditions (one implies the other via J^2 = 1).

    Vectorize: write M = M_R + i M_I where M_R, M_I are real n^2 x n^2 matrices. Express the constraint as a real linear system on the 2n^4 real parameters. Find the kernel.

    **Step 5: Compute the moduli space dimension at general n.**
    Determine dim(ker) as a function of n. If a closed-form formula is obtainable, state it. Otherwise, express as the dimension of the solution space of the derived linear system.

    **Step 6: Cross-check with Barrett's D form.**
    Barrett 2015 Proposition 3.1 gives D(v tensor X) = sum_I (K_I X + epsilon' X K_I*) tensor gamma_I v for the matrix geometry H = V tensor M_n(C) with V = C^2. For KO-dim 6, epsilon' = +1. Identify the gamma matrices on V = C^2 and count the free parameters K_I. Verify the dimension matches Step 5.

    **Step 7: Explicit construction at n=2.**
    For n=2: H = C^8, M is 4x4 complex (32 real parameters). Build J_+, J_- explicitly as 4x4 matrices. Solve the constraint system. Write down an explicit basis for the moduli space.

    Include ASSERT_CONVENTION at top of file.
  </action>
  <verify>
    1. Dimensional consistency: H_+ and H_- each have dimension n^2; M is n^2 x n^2 complex; D is 2n^2 x 2n^2
    2. n=1 limiting case: H = C^2, gamma = diag(1,-1), J = sigma_1 * conj. D = [[0,d],[d,0]] with JD = DJ forcing d real. Moduli dim = 1.
    3. D = 0 is always in the moduli space (trivial solution exists)
    4. J antilinearity properly tracked: JDJ^{-1} involves conj(D), not D
    5. Barrett cross-check: dimension from block decomposition matches Barrett's K_I parameter count
    6. Self-adjointness: D = [[0, M^dagger], [M, 0]] is automatically D* = D for any M (verify this is correct)
    7. Gamma anticommutation: D = [[0, M^dagger], [M, 0]] in the (H_+, H_-) basis gives D gamma + gamma D = 0 (verify)
  </verify>
