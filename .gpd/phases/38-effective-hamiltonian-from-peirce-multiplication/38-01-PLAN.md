  <name>Task 1: Construct and diagonalize 2-site Hamiltonian H_2</name>
  <files>code/effective_hamiltonian.py</files>
  <action>
    Build the 2-site Clifford Heisenberg Hamiltonian and compute its full spectrum.

    ASSERT_CONVENTION at top: natural_units, jordan_product=(1/2)(ab+ba), clifford_normalization={T_a,T_b}=(1/2)delta_{ab}I, coupling_convention=J_gt_0_antiferro, v_half_basis=(x2_0..x2_7,x3_0..x3_7)

    1. Import compute_T_b_matrices() from code/octonion_algebra.py. This returns all 10 T_b as 16x16 real numpy arrays. Do NOT re-derive these operators.

    2. Separate T_{b_1} = (1/4)*I_16 (index 0) from the 9 traceless generators T_a (indices 1..9). Verify T_{b_1} = (1/4)*I_16 to machine precision.

    3. Verify single-site Casimir: C_1 = sum_{a=1}^{9} T_a^2. From {T_a, T_a} = (1/2)*I, each T_a^2 = (1/4)*I, so C_1 = (9/4)*I_16. Check: norm(C_1 - (9/4)*I_16) < 1e-14.

    4. Construct 2-site Hamiltonian on R^{16} tensor R^{16} = R^{256}:
       H_2 = J * sum_{a=1}^{9} (T_a tensor I_16) @ (I_16 tensor T_a)
           = J * sum_{a=1}^{9} numpy.kron(T_a, T_a)
       Set J = 1 (dimensionless lattice units). The sign convention is J > 0 = antiferromagnetic.

    5. Verify H_2 is real symmetric: norm(H_2 - H_2.T) < 1e-14.

    6. Verify Tr(H_2) = 0. Since each T_a is traceless (a=2..10), Tr(T_a tensor T_a) = Tr(T_a)*Tr(T_a) = 0. So Tr(H_2) = 0.

    7. Compute all 36 Spin(9) generators: G_{ab} = [T_a, T_b] = T_a @ T_b - T_b @ T_a for 1 <= a < b <= 9 (using the 9 traceless T_a). Extend to 2-site: G_{ab}^{total} = numpy.kron(G_{ab}, I_16) + numpy.kron(I_16, G_{ab}).

    8. Verify [H_2, G_{ab}^{total}] = 0 for all 36 generators. Report max Frobenius norm across all 36 commutators. Must be < 1e-12.

    9. Diagonalize H_2 using numpy.linalg.eigh (real symmetric). Store all 256 eigenvalues sorted.

    10. Identify eigenvalue degeneracies: group eigenvalues by value (tolerance 1e-10), count multiplicity of each distinct level. Report the spectrum as a list of (eigenvalue, multiplicity) pairs.

    11. Determine ferro/antiferro character: construct the SWAP operator P on R^256 that permutes the two sites (P|v1 tensor v2> = |v2 tensor v1>). The eigenstates of P have eigenvalue +1 (symmetric sector, dim 136) or -1 (antisymmetric sector, dim 120). Project the ground state eigenvector(s) onto symmetric and antisymmetric sectors. If the ground state is in the antisymmetric sector, the system is antiferromagnetic. If symmetric, ferromagnetic.

    12. Compute the energy gap: Delta = E_1 - E_0 (first excited level minus ground state).

    Implement as functions: construct_2site_hamiltonian(J=1.0), diagonalize_2site(H), verify_spin9_commutation(H, T_matrices), analyze_spectrum(eigenvalues, eigenvectors), determine_magnetic_character(eigenvectors, ground_indices).
  </action>
  <verify>

    2. Tr(H_2) = 0 to machine precision
    3. Single-site Casimir: sum T_a^2 = (9/4)*I_16, error < 1e-14
    4. Spin(9) commutation: max |[H_2, G_{ab}]|_F < 1e-12 for all 36 generators
    5. Eigenvalue count: exactly 256 eigenvalues (counting multiplicity)
    6. Sum of eigenvalues = 0 (trace zero)
    7. All eigenvalues real (no imaginary parts beyond machine epsilon)
    8. Degeneracy structure: multiplicities sum to 256
    9. Ferro/antiferro determination: ground state sector unambiguously identified
    10. J=0 limit: if J is set to 0, all 256 eigenvalues should be 0
  </verify>

       The Heisenberg coupling C_2 = sum T_a tensor T_a is related to the SWAP operator P by: P = 2*C_2 + constant (from the identity T_a tensor T_a = (1/2)(T_a tensor I + I tensor T_a)^2 - (1/2)(sum T_a^2 tensor I + I tensor sum T_a^2)). Use this to cross-check eigenvalues.

    2. Compute the eigenvalue on each irrep using the Casimir relation. For the Heisenberg coupling:
       C_2^{(12)} = sum T_a^(1) T_a^(2) = (1/2)(C_2^{total} - C_2^{(1)} - C_2^{(2)})
       where C_2^{total} = sum (T_a^(1)+T_a^(2))^2 and C_2^{(i)} = sum T_a^{(i)2} = (9/4)*I.
