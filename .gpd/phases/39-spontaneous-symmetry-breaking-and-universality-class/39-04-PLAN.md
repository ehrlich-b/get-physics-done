    **UC1: Gapless excitations**
    - Statement: The H_eff on Z^d (d >= 3) has gapless excitations in the thermodynamic limit.
    - Proof: Goldstone's theorem applied to SSB of continuous symmetry Spin(9) -> Spin(8).
    - Conditions: (a) Continuous symmetry Spin(9) (dim 36) -- verified Phase 38. (b) SSB to Spin(8) (dim 28) -- proved Plan 01. (c) d >= 3 so SSB occurs -- Plan 01. (d) Short-range interactions -- nearest-neighbor on Z^d.

    - NOTE: UC1 requires gapless excitations, not that all modes are linear. Even Type-B modes (omega ~ k^2) are gapless. UC1 is satisfied regardless of the Goldstone type.

    **UC2: Algebraic correlation decay**
    - Statement: Two-point correlation function decays as a power law |C(r)| ~ r^{-alpha} at large distances.
    - Proof: From massless Goldstone propagator in the NL sigma model (Plan 03).
    - TWO CASES depending on Goldstone type from Plan 02:
      a) Type-A (linear dispersion omega = c_s |k|): C(r) = int d^dk e^{ikr}/(c_s^2 k^2) ~ r^{-(d-2)}. For d=3: C(r) ~ 1/r.
      b) Type-B (quadratic dispersion omega = D k^2): The propagator is 1/(i omega - D k^2) in frequency-momentum space. The equal-time correlator is still C(r) ~ int d^dk e^{ikr}/k^2 ~ r^{-(d-2)} because the equal-time correlator depends on the STATIC structure factor, not the dynamic one. The static propagator for Type-B modes is STILL 1/k^2 (from the constraint of the sigma model target).
    - IMPORTANT SUBTLETY: Even for Type-B Goldstone modes, the STATIC (equal-time) correlator decays as r^{-(d-2)}, same as Type-A. The difference between Type-A and Type-B appears in the DYNAMIC correlator (time-dependent), not the static one. For UC2, which concerns spatial correlation decay, both types give the same power law.
    - Verify: for the O(N) model, the transverse susceptibility chi_perp(k) ~ 1/k^2 for both ferro and antiferro, giving C(r) ~ r^{-(d-2)} in either case.
    - Result: UC2 satisfied with alpha = d - 2 for d >= 3.

    **UC3: Isotropy (emergent rotational invariance)**
    - Statement: The continuum limit of the lattice sigma model has full O(d) rotational invariance (not just the Z_d cubic lattice symmetry).
    - Proof: Cubic lattice anisotropy is an irrelevant perturbation in the RG sense.
    - For the O(N) model on Z^d, the leading cubic anisotropy operator has scaling dimension d + O(1/N) at the Goldstone fixed point. For N = 9 (large), the anisotropy is safely irrelevant.
    - Cite v9.0 Phase 34 Hasenbusch argument: for the O(3) model, the cubic anisotropy exponent rho ~ 2.02 > d = 3, making it irrelevant. For O(N) with N >= 3, the result is the same or stronger (anisotropy MORE irrelevant for larger N due to larger symmetry group).
    - For N = 9: the O(9) model has an even larger symmetry group than O(3), so anisotropy is even more irrelevant. No new computation needed -- the Hasenbusch argument extends.
    - Result: UC3 satisfied.

    **UC4: Osterwalder-Schrader reflection positivity**
    - Statement: The lattice model satisfies OS reflection positivity, enabling Wick rotation and connection to Minkowski signature.
    - Proof: From DLS framework (Plan 01).
    - Conditions: (a) Bipartite lattice Z^d -- proved Phase 38. (b) Inner-product interaction H = J sum n_i . n_j -- bilinear in T_a (Phase 38). (c) On-site measure positive and symmetric -- Haar measure on S^8 or the quantum analog.
    - For the CLASSICAL model: RP is established as part of the FSS infrared bound proof (Plan 01). The classical S^8 model on Z^d with inner-product interaction satisfies RP by the standard DLS construction.
    - For the QUANTUM model: This is the Speer subtlety. If BCS lifts to quantum without requiring quantum RP, then RP is a property of the classical model only, and OS positivity comes from the classical -> quantum correspondence. If quantum RP is required independently, cite the Plan 01 analysis.
    - OS reconstruction: RP on the lattice -> OS positivity in the continuum -> Wick rotation to Lorentzian signature. This is the standard OS reconstruction theorem.
    - Result: UC4 satisfied (from classical RP in DLS construction; quantum model inherits via BCS or is conditional).
