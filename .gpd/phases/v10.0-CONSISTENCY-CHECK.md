## Formula Inconsistency in 39-goldstone-modes.md

**Finding (MINOR, NON-PROPAGATING):** In derivations/39-goldstone-modes.md, line 226, the spin-wave velocity formula is written as:

```
c_s = sqrt(2 J rho_s / chi_perp)
```

This has an extra factor of `2J` compared to the standard hydrodynamic formula `c_s^2 = rho_s / chi_perp` used in:
- derivations/39-sigma-model.md (line 126)
- derivations/41-o9-quantitative-results.md (Eq. 41.4)

The estimate on the next line, `c_s = Ja * sqrt(2/N) = Ja * sqrt(2/9) ~ 0.47 Ja`, is also inconsistent with the correct Phase 41 value `c_s = J*sqrt(3/2) = 1.225 Ja`.

**Impact assessment:** LOW -- NON-PROPAGATING. The erroneous formula and estimate in 39-goldstone-modes.md are in a brief remark about the spin-wave velocity (Section 4.2, a 3-line aside). They are NOT used in any downstream calculation. The correct formula is established in 39-sigma-model.md and used definitively in Phase 41. No downstream artifact references the 39-goldstone-modes formula for c_s. The value `sqrt(2/9)` does not appear anywhere outside that file.

**Root cause:** The formula appears to be from a different convention where rho_s absorbs a factor of J*S^2 (the Heisenberg spin stiffness convention `rho_s = JS^2/(N-1)` rather than the unit-vector convention `rho_s = J/(N-1)`). The aside was likely written before the sigma model normalization was finalized in Plan 03.

**Suggested fix:** Correct the formula in 39-goldstone-modes.md Section 4.2 to `c_s = sqrt(rho_s / chi_perp)` and update the estimate, or add a note that the numerical value is computed properly in Phase 41.

## Correlation Function Consistency

**Checked:** Phase 39 Plan 04 (Eq. 39.11) gives the connected correlation as:
```
C(r) ~ Gamma(d/2-1) / (4*pi^(d/2) * rho_s * r^(d-2))
```

Phase 41 (Eq. 41.21) gives the TOTAL connected correlator as:
```
C(r) = (N-1) * Gamma(d/2-1) / (4*pi^(d/2) * rho_s * r^(d-2))
```

**Assessment:** These are CONSISTENT. Phase 39 writes C(r) without the explicit (N-1) factor because in the context of UC2 verification, the power-law decay exponent is the relevant quantity (it is the same regardless of the N-1 prefactor). Phase 41 correctly computes the full numerical prefactor by summing over all N-1 = 8 Goldstone mode contributions. The d=3 result C(r) = 16/(pi*J*r) is verified by explicit substitution:

- N-1 = 8, Gamma(1/2) = sqrt(pi), pi^(3/2) = pi*sqrt(pi)
- C(r) = 8*sqrt(pi) / (4*pi*sqrt(pi) * J/8 * r) = 8*8 / (4*pi*J*r) = 16/(pi*J*r). **CORRECT.**
