  <name>Task 1: Exhaustive SymPy S4 testing on M_2(C)^sa</name>
  <files>code/sp_verification.py</files>
  <action>
    Add comprehensive S4 testing to the verification harness.

    **S4 statement (arXiv:1803.11139 Def. 2):**
    If a & b = 0 then b & a = 0.

    **Test 1 -- Sharp effects with phi = identity:**
    On M_2(C)^sa, parametrize all rank-1 projections p = |v><v| by Bloch sphere angles (theta, phi_angle). For each pair (p, q) where p & q = 0 (which means C_p(q) = pqp = 0, i.e., p and q are orthogonal projections), check q & p = 0.
    For orthogonal projections in M_2(C)^sa, p perp q iff q = p^perp = 1 - p. Check: C_q(p) = qpq. Since q = 1 - p and p is rank 1, qpq = (1-p)p(1-p) = 0. So S4 holds trivially for sharp orthogonal effects.
    This is expected -- the real test is general (non-sharp) effects.

    **Test 2 -- General effects with phi = identity:**
    Parametrize general effects a = (1/2)(I + r_a . sigma) where |r_a| <= 1.
    For pairs (a, b) where a & b = 0, check b & a = 0.
    Finding pairs with a & b = 0 for general effects is non-trivial. Strategy: fix a, then solve C_a(b) = 0 for b (this is a linear equation in the components of b). For each such b, check C_b(a) = 0.

    Using the spectral extension: a = lambda_1 p_1 + lambda_2 p_2, then a & b = lambda_1 C_{p_1}(b) + lambda_2 C_{p_2}(b). For a & b = 0: lambda_1 p_1 b p_1 + lambda_2 p_2 b p_2 = 0.

    **Test 3 -- Different phi choices:**
    (i) phi = identity (dim M = dim B): most favorable case
    (ii) phi = coarse-graining: e.g., B is 4-dim (M_2(C)^sa), M is 2-dim (R^2 simplex), phi maps B's effects to classical effects on M. This is where S4 is most likely to fail.
    (iii) phi = general positive unital compression-preserving map

    **Test 4 -- Positive/negative controls:**
    Luders product: S4 should PASS (positive control)
    Matrix multiplication: S4 should FAIL (negative control, from Plan 01)

    Report for each phi choice: "S4: PASS (all tested pairs)" or "S4: FAIL (counterexample: a = ..., b = ...)"
  </action>
  <verify>
    1. Sharp orthogonal effects: S4 holds trivially (both compressions annihilate)
    2. General effects: either all tested pairs pass, or explicit counterexample found
    3. Positive control (Luders): S4 passes
    4. Negative control (matrix mult): S4 fails
    5. Multiple phi choices tested with clear results for each
    6. Effects tested in general position (not just basis-aligned -- the false-progress red flag)
  </verify>

<task type="auto">
  <name>Task 2: General S4 proof or disproof</name>
  <files>derivations/04-axiom-S4.md</files>
  <action>
    Based on the SymPy results from Task 1, construct a general proof or exhibit a general disproof of S4.

    **IF Task 1 supports S4 (all tested pairs pass):**

    Attempt a general proof. Three routes from RESEARCH.md:

    Route A -- Orthogonality preservation:
    Show phi preserves orthogonality: if a perp b (i.e., a & b = 0), then phi(a) perp phi(b). If compressions on M commute when their generating effects are orthogonal under phi, then:
    a & b = 0 means C_{phi(a)}(b) = 0 (no compression of b by phi(a)'s face)
    Need: C_{phi(b)}(a) = 0 (no compression of a by phi(b)'s face)
    This requires phi to map orthogonal effects to orthogonal faces in M.

    Route B -- Inner product symmetry (arXiv:1803.08453):
    Construct an inner product <.,.> on V such that <a & b, c> = <b, a & c> for all a, b, c. If such an inner product exists, S4 follows: <a & b, 1> = <b, a & 1> = <b, a>, and if a & b = 0 then <b, a> = 0. Then <b & a, 1> = <a, b> = <b, a> = 0 implies b & a = 0 (by positivity of &).
    This route requires constructing the inner product from OUS primitives (no Hilbert space trace!).

    Route C -- Faithfulness:
    If phi is isometric (injective on states), orthogonality relations are preserved bidirectionally. Formally: if phi is a unital order embedding, then face(phi(a)) in M captures exactly the states that are supported on face(a) in B. Then C_{phi(a)}(b) = 0 implies the faces are complementary, and by injectivity, C_{phi(b)}(a) = 0.

    Identify which route succeeds (or fails) and why. The successful proof must:
    - Be longer than the combined S1-S3 proofs (red flag check)
    - Reference specific properties of phi (not generic for any SP)
    - Handle both sharp and general effects
    - Address the spectral extension explicitly

    **IF Task 1 reveals S4 failure:**

    Exhibit the counterexample formally:
    - Specify B, M, phi, and the effects a, b explicitly
    - Show a & b = 0 (exact symbolic computation)
    - Show b & a != 0 (exact symbolic computation, exhibit the nonzero entry)
    - Identify which property of phi is responsible for the failure
    - State whether S4 holds for a restricted class of phi (e.g., isometric phi only)

    **IF S1-S7 all hold:**

    Invoke van de Wetering's Theorem 1:
    "A finite-dimensional sequential product space is order-isomorphic to a Euclidean Jordan algebra."
    Cite the theorem (do NOT re-derive). Identify the EJA type for the qubit case: should be M_2(C)^sa (the 4-dimensional EJA of 2x2 complex Hermitian matrices, isomorphic to the spin factor V_3).

    State the conclusion clearly:
    - If S1-S7 hold for all valid phi: "The self-modeling sequential product defines an EJA structure on the effect space. Self-modeling alone suffices."
    - If S1-S7 hold for constrained phi: "The self-modeling sequential product defines an EJA structure when phi satisfies [constraint]. The constraint has physical interpretation [interpretation]."
    - If S4 fails for all phi: "S4 fails. The self-modeling construction does not yield a sequential product space. Proceed to SPFM-05 (Plan 05) and Phase 7 (D'Ariano backup)."
  </action>
