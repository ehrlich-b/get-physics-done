  <name>Task 1: Define the compression-based sequential product and resolve the framing</name>
  <files>derivations/04-sequential-product-definition.md</files>
  <action>
    Define the self-modeling sequential product on a finite-dimensional order unit space using ONLY order unit space primitives (order structure, effect space [0,1]_V, compression maps C_p, tracking map phi). No Hilbert space imports.

    **Step 1 -- Quote axiom definitions verbatim.**
    Fetch arXiv:1803.11139 and quote Definition 2 (S1-S7) exactly. Also quote Corollary 7 (spectral decomposition) and the relevant compression properties from Niestegge (2008): C_p is a positive idempotent map, C_p(1) = p, C_p C_q = C_q C_p when p <= q.

    **Step 2 -- Define the product for sharp effects.**
    For sharp effects p, q in E(B):
    - E(B) framing: a & b = C_a(b), where C_a is the Alfsen-Shultz compression for face(a). The self-modeling constraint enters through the feedback loop: testing a on B triggers compression C_{phi(a)} on M, which updates M's state; the updated M-state then determines the context for testing b on B. Formalize this as: a & b encodes "the effect of first testing a, then testing b, where the intermediate model update via phi modifies the state but the product is defined on E(B)."
    - E(B x M) framing: define the product on the composite effect space. Analyze whether this yields a sequential product on E(B x M) satisfying vdW's axioms (which require a single system, not a composite).

    **Step 3 -- Analyze both framings.**
    For each framing, determine:
    (a) Is the product well-defined (maps effects to effects)?
    (b) Is the product bilinear?
    (c) Does S3 (1 & a = a) hold?
    (d) Does the framing match vdW's single-system setup?
    Document the selected framing and the rejected framing's failure mode.

    **Step 4 -- Extend to general effects via spectral decomposition.**
    For general effect a = sum_i lambda_i p_i (spectral decomposition, Corollary 7), define:
    a & b = sum_i lambda_i (p_i & b)
    Prove this is well-defined (independent of decomposition choice) and bilinear.
    For the second argument, linearity follows from linearity of compressions.
    For the first argument, linearity follows from the spectral extension IF the decomposition is unique.

    **Step 5 -- Verify classical limit.**
    When B is a simplex (state space is a probability simplex), show a & b reduces to pointwise multiplication a_i * b_i. This is the Gudder-Greechie uniqueness benchmark. Work this out explicitly for n = 2 (bit) and n = 3.

    **Step 6 -- Circularity audit.**
    Review every step of the construction. Verify NO occurrence of: sqrt(a), trace, inner product, density matrices, Luders rule a & b = sqrt(a) b sqrt(a), or any C*-algebra structure. Every operation must be expressible using only (1) order structure, (2) effect space [0,1]_V, (3) compression maps C_p, (4) tracking map phi. List each primitive used and its OUS justification.

    **Step 7 -- State the parametrization by phi.**
    The sequential product is defined for a class of valid tracking maps phi: E(B) -> E(M). Do NOT pin phi uniquely. State what properties phi must satisfy for the product to be well-defined (positive, unital, compression-preserving). Record that axiom verification (Plans 02-04) will reveal what additional properties phi needs.

    Use "per user decision" annotations where locked decisions constrain the derivation.
  </action>

    2. Well-definedness: a & b in [0,1]_V whenever a, b in [0,1]_V (positivity of compressions + norm bounds)
    3. Bilinearity: linearity in each argument demonstrated formally
    4. S3 check: 1 & a = a holds (requires phi(1) = 1_M or equivalent)
