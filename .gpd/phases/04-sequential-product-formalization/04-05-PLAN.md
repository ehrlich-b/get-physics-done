    Starting from the explicit S4 counterexample produced in Plan 04:

    **Step 1 -- Analyze the counterexample.**
    Take the counterexample (a, b) with a & b = 0 but b & a != 0 from Plan 04.
    Compute: what does a & b = 0 mean in terms of compressions? C_{phi(a)}(b) = 0 means b lies in the kernel of the compression for phi(a)'s face.
    Compute: what does b & a != 0 mean? C_{phi(b)}(a) != 0 means a does NOT lie in the kernel of the compression for phi(b)'s face.

    **Step 2 -- Identify the asymmetry.**
    The S4 failure means: face(phi(a)) annihilates b, but face(phi(b)) does NOT annihilate a. This is an asymmetry in how phi maps B's orthogonality structure to M's face structure.
    Formalize: define "phi preserves orthogonality" as: a perp b implies phi(a) perp phi(b) (in M's effect space). Does the self-modeling phi satisfy this?
    If not: which specific pair's orthogonality is not preserved, and why?

    **Step 3 -- Vary phi systematically.**
    From Plan 04's phi-dependence data:
    - For which phi class does S4 hold? (e.g., isometric/injective phi?)
    - For which phi class does S4 fail? (e.g., coarse-graining phi?)
    State the boundary as a formal condition: "S4 holds iff phi satisfies C, where C is: ..."

    **Step 4 -- Physical interpretation.**
    Interpret the condition C physically in the self-modeling framework:
    - If C = "phi is injective on states": the model must faithfully track all distinguishable states of B
    - If C = "phi preserves orthogonality": the model must preserve the distinction between complementary tests
    - If C = "phi is an order embedding": the model must preserve the full order structure of B
    State which of these corresponds to a physically reasonable constraint on self-models.
