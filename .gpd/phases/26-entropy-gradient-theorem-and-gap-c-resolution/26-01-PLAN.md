    **Section 2: Route B (Boundary Argument).**
    A second independent route from finite-block structure:
    1. A self-modeling block has finite Hilbert space dim d_B * d_M [Paper 5 + finite lattice]
    2. Entropy is bounded: 0 <= S(rho_BM) <= ln(d_B * d_M) [finite-dim QM]
    3. Under iterated SWAP interactions, entropy is non-decreasing: S_{N+1} >= S_N [Phase 23, Lindblad H-theorem for unital channel E with rho_M = I/2]
    4. Bounded + non-decreasing + S < S_max at some time t -> there exists t_0 < t with S(t_0) < S(t) [arithmetic: if S never decreases and hasn't reached max, trace backward to find lower S]
    5. The block must have a low-entropy boundary condition: S(t_initial) < S_max

    This route does NOT use the Landauer bound directly. It uses only finitude + monotonicity. The logical structure is: finite + monotonic + not-yet-equilibrated -> low-entropy past. This is essentially Boltzmann's argument for the arrow of time, but now connected to self-modeling via the entropy monotonicity from Phase 23.

    **Section 3: Route C (rho Selection).**
    A third route using the experiential measure directly:
    1. The experiential density rho is defined on blocks of structure space [Paper 5]
    2. At thermal equilibrium, I(B;M) = 0 and rho = 0 [Phase 25-01, proved analytically]
    3. Blocks with rho > 0 must have I(B;M) > 0, hence are non-equilibrium [contrapositives]
    4. Among all blocks of structure space, rho selects those with self-modeling -- which requires non-equilibrium -- which requires an entropy gradient
    5. Therefore: the rho-weighted ensemble is concentrated on blocks with an entropy gradient. The "Past Hypothesis" is a selection effect within structure space: blocks that contribute to rho necessarily have low-entropy pasts.

    This route is the weakest-assumption version: it does NOT require the Landauer bound or the monotonicity theorem individually, only the equivalence rho > 0 <-> I > 0 <-> non-equilibrium. It reframes the Past Hypothesis as an observer selection effect in structure space, analogous to the anthropic principle but more precise (quantitative via the Landauer bound).
