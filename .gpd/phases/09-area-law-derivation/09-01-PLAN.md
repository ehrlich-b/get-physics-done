  <name>Task 1: WVCH thermal mutual information area law for self-modeling Hamiltonian</name>
  <files>derivations/09-wvch-thermal-area-law.md</files>
  <action>
    Derive the WVCH thermal mutual information bound applied to the self-modeling Hamiltonian H = sum_{<x,y> in E} h_{xy} with h_{xy} = JF (SWAP operator) on a graph G = (V, E).

    **Part A: State the WVCH theorem (Wolf-Verstraete-Cirac-Hastings 2008)**

    Theorem (WVCH): For a local Hamiltonian H = sum_X Phi(X) on a lattice, the Gibbs state rho_beta = exp(-beta*H)/Z satisfies:

    I(A:B) <= 2*beta * sum_{X: X intersects both A and B} ||Phi(X)||

    where I(A:B) = S(A) + S(B) - S(AB) is the mutual information between region A and its complement B = V\A.

    Cite: Wolf et al. 2008, PRL 100, 070502 (arXiv:0704.3906).

    **Part B: Verify ALL hypotheses for the self-modeling Hamiltonian**

    Check each hypothesis of WVCH against the Phase 8 results:

    (H1) H is a sum of local terms: YES. H = sum_{<x,y>} h_{xy} is a nearest-neighbor Hamiltonian on graph G. Interaction range = 1 edge. Reference: derivations/08-lattice-definition.md, Definition 5.

    (H2) The state is a Gibbs state rho_beta = exp(-beta*H)/Z at some finite temperature T = 1/beta: This is an ASSUMPTION to be stated as the gap. The self-modeling framework does not by itself specify a global quantum state. The identification of the physically relevant state as a thermal state is a physical argument -- to be developed in Plan 03. For now, ASSUME the state is thermal and derive the consequence.

    (H3) Each interaction term has finite operator norm: ||h_{xy}|| = ||JF|| = |J| * ||F|| = |J| (since SWAP is unitary, ||F|| = 1). Reference: Phase 8, Eq. (08-03.1).

    (H4) The interaction is nearest-neighbor (finite-range): YES. h_{xy} is supported on single edges <x,y> in E. Reference: derivations/08-lattice-definition.md, Theorem 1 + derivations/08-hamiltonian-construction.md.

    **Part C: Apply the WVCH bound to the self-modeling Hamiltonian**

    For a bipartition V = A union B, the boundary terms are those h_{xy} with x in A, y in B (or vice versa). Each boundary term has ||h_{xy}|| = |J|. The number of boundary terms is |boundary(A)| = |{<x,y> in E : x in A, y in B}|.

    Therefore: I(A:B) <= 2*beta*|boundary(A)|*|J|

    This is an area law for mutual information: the bound scales with the boundary area |boundary(A)|, not the volume |A|.

    Note: The bound is sign-independent -- it depends on |J|, not sign(J). Both FM and AFM give the same MI bound.

    **Part D: Dimensional analysis**

    Check: [I(A:B)] = [dimensionless] (entropy is dimensionless in nats). [beta] = [1/energy]. [|J|] = [energy]. [|boundary|] = [dimensionless count]. So [2*beta*|boundary|*|J|] = [dimensionless]. Consistent.

    **Part E: Interpretation and limitations**

    1. This bounds MUTUAL INFORMATION, not von Neumann entropy S(A). For the thermal state, S(A) has a volume-law extensive contribution S(A) ~ |A|*s(T) from the thermal entropy density s(T). The mutual information I(A:B) = S(A) + S(B) - S(AB) subtracts out the extensive parts, leaving only the boundary correlation.

    2. For pure states (T=0), I(A:B) = 2*S(A), so the MI area law IS an entropy area law. But the ground state may not be a Gibbs state in the strict T -> 0 limit (the limit is singular for gapless systems).

    3. The WVCH bound tightens as T increases (beta decreases): at high T, the thermal state approaches the maximally mixed state with I -> 0. The bound is tightest (largest, most informative) at low T (large beta).

    4. State the gap explicitly: "The WVCH bound applies to the Gibbs state. The identification of the self-modeling fixed-point state with a thermal state is an assumption (Assumption A1). This assumption is physically motivated because [developed in Plan 03] but is not derived from the self-modeling axioms alone."

    **Part F: Higher dimensions**

    The WVCH bound works in ALL spatial dimensions with no modification. For the self-modeling Hamiltonian on Z^d: I(A:B) <= 2*beta*|boundary_d(A)|*|J| where |boundary_d(A)| is the number of nearest-neighbor bonds crossing the boundary in d dimensions. This automatically gives the correct area-law scaling: |boundary_d(A)| ~ L^{d-1} for a hypercubic region of linear size L.

    Do NOT overclaim: the WVCH result is for mutual information of thermal states, not for S(A) of the ground state.

  <name>Task 2: Heisenberg ground-state entanglement characterization for both signs of J</name>
  <files>derivations/09-heisenberg-entanglement.md</files>
  <action>
    Provide a complete characterization of the ground-state entanglement structure of the Heisenberg model h_{xy} = (J/2)(sigma.sigma) for both signs of J. This serves as the validation anchor and honest accounting for Plan 03.

    **Part A: Ferromagnetic case (J < 0)**

    1. Ground state: The fully polarized state |up up ... up> (and SU(2) rotations thereof). This is a product state.
    2. Degeneracy: (N+1)-fold for N spin-1/2 sites (total spin S = N/2 multiplet). All degenerate ground states are product states (coherent spin states) or superpositions within the S=N/2 sector.
    3. Spectral gap: Delta ~ O(J/N^2) from quadratic magnon dispersion E_k ~ J*k^2 for small k. In the thermodynamic limit N -> infinity, the gap closes: Delta -> 0. The FM Heisenberg model is gapless.
    4. Entanglement: S(A) = 0 for any subsystem A when the state is a fully polarized product state. Even in the degenerate manifold, the maximally entangled superposition has S(A) ~ ln(min(|A|, N-|A|) + 1) ~ ln(N), which is at most logarithmic in system size. This is much less than volume-law.
    5. Conclusion: FM case trivially satisfies area law (in fact, S=0 for the canonical ground state). But this is physically uninteresting for gravity: a product state has zero entanglement, so it cannot drive Jacobson's entanglement equilibrium argument.

    **Part B: Antiferromagnetic case (J > 0)**

    1. Ground state: The unique singlet state (total spin S=0), obtainable via Bethe ansatz in 1D. Highly entangled.
    2. Spectral gap: Exactly gapless for spin-1/2 in 1D (Bethe ansatz). The low-energy physics is described by the SU(2)_1 Wess-Zumino-Witten (WZW) conformal field theory with central charge c = 1.
    3. Entanglement (1D): Calabrese-Cardy formula gives S(L) = (c/3)*ln(L/a) + const for a block of L sites in a chain of length N >> L, with c = 1. So S(L) = (1/3)*ln(L) + const. This is a LOGARITHMIC VIOLATION of strict area law.
    4. For the open chain of length N: S(L) = (c/6)*ln((2N/pi)*sin(pi*L/N)) + const (Calabrese-Cardy for open BC). For periodic BC: S(L) = (c/3)*ln((N/pi)*sin(pi*L/N)) + const.
    5. Higher dimensions (D >= 2): Numerical evidence (Kallin et al. 2011, Kulchytskyy et al. 2015) shows area-law scaling S(A) ~ c_1*|boundary(A)| + c_2*log + O(1) for AFM Heisenberg on square and cubic lattices. Not rigorously proven.
    6. Conclusion: AFM ground state has a logarithmic correction to area law in 1D. This is qualitatively much better than volume law (which scales as L), but it is NOT a strict area law.

    **Part C: Why Hastings does NOT apply**

    State explicitly:

    Hastings' 2007 area-law theorem (JSTAT P08024, arXiv:0705.2024) proves that for a 1D gapped Hamiltonian with spectral gap Delta > 0, the ground state satisfies S(A) <= c * xi * log(xi) where xi ~ v_LR/Delta is the correlation length. The theorem requires:

    (H1) 1D lattice -- our system is on Z^1 for the 1D case. OK.
    (H2) Local interactions -- our h_xy is nearest-neighbor. OK.
    (H3) Spectral gap Delta > 0 -- FAILS for both signs of J. FM: Delta ~ J/N^2 -> 0. AFM: Delta = 0 exactly (gapless Bethe ansatz spectrum).

    Since hypothesis (H3) fails, Hastings' theorem is inapplicable. The logarithmic violation S ~ (1/3)ln(L) for AFM is precisely the expected behavior for a 1D gapless system described by a c=1 CFT.

    Also note: Brandao-Horodecki (2013/2015) requires exponential correlation decay, which fails for both cases. FM has long-range order (power-law correlations). AFM has algebraic correlation decay <S_0.S_r> ~ (-1)^r / r (not exponential).

    **Part D: Summary table**

    Compile a table:

    | Property | FM (J<0) | AFM (J>0) |
    |---|---|---|
    | Ground state | product |up...up> | Bethe ansatz singlet |
    | Degeneracy | N+1 | 1 |
    | Gap (1D) | O(J/N^2) -> 0 | 0 (exact) |
    | S(A) in 1D | 0 | (1/3)ln(L) + const |
    | Hastings applies? | No (gapless) | No (gapless) |
    | Area law? | Trivial (S=0) | Log-corrected |
    | Useful for Jacobson? | No (no entanglement) | Possibly (has entanglement, but log not area) |

    **Part E: Implications for Phase 9**

    Neither sign of J gives a clean area-law result from ground-state analysis alone. This motivates:
    1. WVCH thermal route (Plan 01, Task 1): works for both signs, any T > 0
    2. Channel capacity route (Plan 02): pure-state information-theoretic bound
    3. The "which state?" resolution (Plan 03): the physically relevant state for Jacobson is NOT necessarily the ground state
