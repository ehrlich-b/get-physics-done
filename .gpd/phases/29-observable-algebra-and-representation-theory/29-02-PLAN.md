    **1. express_ju_as_clifford_polynomial(gamma_matrices, J_u)**

    Using the grade decomposition coefficients from compute_grade_decomposition(J_u, gamma_matrices):
    - Write J_u = sum_{S subset {1,...,9}} c_S * gamma_S where gamma_S = ordered product of gamma_i for i in S.

    **2. test_ju_uniqueness(gamma_matrices, J_u)**

    Determine how many elements X in M_16(R) share J_u's defining algebraic properties.

    The defining properties of J_u (from Plan 01 results) depend on what the anticommutation test reveals. Branch on the Plan 01 result:

    **Case A: J_u anticommutes with all 9 gamma_i.**

    Then J_u is characterized by: (a) X^2 = -I, (b) {X, gamma_i} = 0 for all i, (c) X is real 16x16.

    Uniqueness argument: Any X satisfying (b) must be proportional to the volume element omega times J_u (since the centralizer of Cl(9,0) on an irreducible representation is R, but J_u is NOT in the centralizer -- it anticommutes). More precisely: the space of elements anticommuting with all 9 gamma_i in End(R^16) is 0-dimensional if omega = +/-I on R^16 (because the Clifford algebra acts irreducibly and the centralizer is R).

    Wait -- if omega = +I on R^16, then the grade-9 element (omega) acts as +I, and any element in the "odd" part of the algebra (odd total grade when both factors of Cl(9,0) are considered) anticommutes with the generators. But on R^16 the representation factors through one simple component...

    This analysis is subtle. Compute numerically instead:
    - Construct the system of equations: X @ gamma_i + gamma_i @ X = 0 for all i (9 equations, each 16x16 = 256 constraints).
    - Flatten X into a 256-dim vector. Each anticommutation equation gives 256 linear constraints. Total: 9 * 256 = 2304 constraints on 256 unknowns. But many are redundant.
    - Solve for the nullspace of this linear system. The dimension of the nullspace is the number of free parameters.
    - Within the nullspace, impose X^2 = -I (quadratic constraint). Count solutions up to sign.

    **Case B: J_u does NOT anticommute with all 9 gamma_i.**

    Then J_u's defining properties are: (a) X^2 = -I, (b) the specific commutation/anticommutation pattern with gamma_i from Plan 01.
    - Construct the linear system from whatever pattern was found: {X, gamma_i} = 0 for those i where J_u anticommutes, [X, gamma_i] = 0 for those where it commutes, and the actual relation X gamma_i + gamma_i X = M_i for "neither" cases.
    - Solve for the nullspace and characterize.

    In either case, return: {"nullspace_dim": dimension of the linear solution space, "solutions_with_j_squared_minus_i": count of sign-inequivalent solutions (up to X -> -X), "is_unique": True if count = 1}.

    **3. compute_gsm_commutant(gamma_matrices, J_u)**

    Compute the commutant of J_u within the Lie algebra spin(9):
    - spin(9) is spanned by the 36 elements [gamma_i, gamma_j] / 2 for i < j (from Phase 28).
    - For each basis element L_{ij} = (gamma_i @ gamma_j - gamma_j @ gamma_i) / 2, compute [J_u, L_{ij}] = J_u @ L_{ij} - L_{ij} @ J_u.
    - Find the subspace of spin(9) that commutes with J_u: {L in spin(9) : [J_u, L] = 0}.
    - Compute its dimension. This is dim(g_SM) = dim(Lie(G_SM)).
    - Return: {"commutant_dim": dimension, "commutant_basis": list of (i,j) pairs spanning the commutant, "expected_gsm_dim": 12 (for SU(3) x SU(2) x U(1) = 8+3+1)}.

    This cross-checks Krasnov's G_SM = Stab_{Spin(9)}(J_u) and gives an independent verification of the SM gauge group dimension.
  </action>
