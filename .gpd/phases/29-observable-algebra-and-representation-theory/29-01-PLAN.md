    **3. compute_ju_anticommutation(gamma_matrices, J_u)**

    **4. compute_grade_decomposition(target_matrix, gamma_matrices)**

    Decompose an arbitrary 16x16 matrix (in particular J_u) into Clifford grades.

    For Cl(9,0) with generators gamma_1,...,gamma_9, the grade-k basis elements are:
    - Grade 0: I_{16} (1 element)
    - Grade 1: gamma_i (9 elements)
    - Grade 2: gamma_i * gamma_j for i < j (36 elements)
    - Grade k: gamma_{i_1} * ... * gamma_{i_k} for i_1 < ... < i_k (C(9,k) elements)
    - Grade 9: gamma_1 * ... * gamma_9 = omega (1 element)

    Total: sum C(9,k) for k=0..9 = 2^9 = 512 basis elements. But the representation on R^16 maps 512 elements to at most 256 linearly independent 16x16 matrices. Some pairs of Clifford elements map to the same matrix (related by the kernel P_- if omega = +I on R^16).

    Strategy: Enumerate all 2^9 = 512 Clifford monomials using bit-index representation. For each subset S of {1,...,9} (represented as a 9-bit integer), compute gamma_S = ordered product of gamma_i for i in S. Then for each grade k, the grade-k projection of a matrix M is:
    - Collect all monomials of grade k (subsets of size k)
    - Compute the inner products <M, gamma_S> = trace(M^T @ gamma_S) / 16 for each grade-k monomial gamma_S
    - The grade-k projection is: M^{(k)} = sum_S (coefficient_S * gamma_S)

    Since omega = +/-I on R^16, monomials of complementary grades (k and 9-k) are proportional on R^16. This means the 512 monomials give only 256 independent matrices.

    To compute the Clifford expansion coefficients: since the Clifford monomials {gamma_S} satisfy trace(gamma_S^T @ gamma_T) = 16 * delta_{S,T} for |S| + |T| even (up to the omega identification), use:
    - coefficient_S = trace(M^T @ gamma_S) / 16

    Return dict: {"coefficients": dict mapping frozenset(S) -> float, "grade_norms": list of 10 floats (Frobenius norm of grade-k projection for k=0..9), "dominant_grade": grade with largest norm, "reconstruction_error": |M - sum of all projections|}.
