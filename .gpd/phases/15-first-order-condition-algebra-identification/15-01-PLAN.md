  <name>Task 1: Derive first-order condition triviality for Barrett-form D</name>
  <files>derivations/09-first-order-condition.md</files>
  <action>
    Derive [[D_K, L_a], R_b] = 0 for Barrett-form D_K(X) = KX + XK, K in M_n(R)^sym.

    Step 1: Compute [D_1, L_a] on M_n(C).
    D_1(X) = KX + XK = L_K(X) + R_K(X). Then:
    [D_1, L_a](X) = D_1(aX) - a D_1(X) = KaX + aXK - aKX - aXK = (Ka - aK)X = [K,a]X.
    Therefore [D_1, L_a] = L_{[K,a]} (left multiplication by the commutator [K,a]).
    Note: R_K(aX) - aR_K(X) = aXK - aXK = 0 (R_K commutes with L_a).

    Step 2: Compute [[D_1, L_a], R_b] on M_n(C).
    [L_{[K,a]}, R_b](X) = [K,a](Xb) - ([K,a]X)b = [K,a]Xb - [K,a]Xb = 0.
    This is [L_C, R_b] = 0 for C = [K,a], which is the associativity of matrix multiplication.

    Step 3: Verify on the full doubled space H = M_n(C)_p + M_n(C)_ap.
    D = sigma_1 tensor D_1, so D(X_p, X_ap) = (D_1(X_ap), D_1(X_p)).
    pi(a)(X_p, X_ap) = (aX_p, aX_ap) and pi_o(b)(X_p, X_ap) = (X_p b, X_ap b).
    Compute [D, pi(a)](X_p, X_ap) = ([K,a]X_ap, [K,a]X_p) -- this is sigma_1 tensor L_{[K,a]}.
    Then [[D, pi(a)], pi_o(b)](X_p, X_ap) = ([K,a]X_ap b - [K,a]X_ap b, [K,a]X_p b - [K,a]X_p b) = (0, 0).
    The sector-swap does not affect the result because pi_o(b) = R_b acts identically on both sectors.

    Step 4: Conclude A_F = M_n(C) for all K in M_n(R)^sym.
    The first-order condition places no restriction on a. Therefore A_F = M_n(C) (the full algebra).
    The gauge group is U(n): the unitary group of M_n(C).
    At n=4: gauge group = U(4), NOT U(1) x SU(2) x SU(3).

    Step 5: Compare with CCM.
    The CCM classification uses a DIRECT SUM algebra A = M_2(H) + M_4(C), not a simple algebra M_n(C).
    In the direct sum case, D has cross-terms connecting the two summands, and [D, L_a] for a in one summand produces operators that do NOT commute with R_b from the other summand. These cross-term constraints force A_F = C + H + M_3(C).
    For M_n(C) (simple), no cross-terms exist, and [D_K, L_a] = L_{[K,a]} is always a pure left multiplication.

    Step 6: Limiting cases.
    - K = 0: D = 0, first-order condition vacuous, A_F = M_n(C). Consistent.
    - K = lambda * I: [K, a] = 0 for all a, so [D, L_a] = 0, first-order condition trivially 0. A_F = M_n(C). Consistent.
    - K generic non-scalar: [K, a] != 0 for generic a, but L_{[K,a]} still commutes with R_b. A_F = M_n(C). Consistent.

    Step 7: Note on the Pati-Salam fallback.
    The backtracking trigger "If A_F is trivial (C only)" does NOT apply: A_F = M_n(C) is MAXIMAL, not trivial.
    The Pati-Salam fallback (CCSV 2013) concerns DROPPING the first-order condition entirely, which introduces quadratic terms in the inner fluctuations. Here, the first-order condition is trivially SATISFIED, which is mathematically distinct from dropping it.

    Document all results with equation numbers following the convention (15-01.N).
  </action>
  <verify>
    1. [D_1, L_a] = L_{[K,a]}: verify by expanding D_1(aX) - aD_1(X) = KaX + aXK - aKX - aXK = [K,a]X. The R_K part must cancel: aXK - aXK = 0.
    2. [L_C, R_b] = 0: verify by expanding C(Xb) - (CX)b = CXb - CXb = 0 (associativity). This is the same identity used in Phase 13 for order zero.
    3. Doubled space: verify [D, pi(a)] = sigma_1 tensor L_{[K,a]} by checking both components.
    4. Limiting cases K=0 and K=I both give A_F = M_n(C).
    5. The argument is structurally independent of K: it uses only that [D_1, L_a] is a left multiplication, which holds for ANY D_1 = L_K + R_K.
  </verify>
  <done>First-order condition for Barrett-form D derived: [[D_K, L_a], R_b] = 0 for all a, b, K. A_F = M_n(C) (full algebra). Gauge group U(n). CCM comparison documented. All limiting cases checked.</done>
