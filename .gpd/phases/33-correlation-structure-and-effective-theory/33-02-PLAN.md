  <name>Task 1: 2D Heisenberg ED -- ground state, correlations, and sublattice structure</name>
  <files>code/correlation_2d.py, data/correlation/corr_2d_4x4.json</files>
  <action>
    Build 2D correlation analysis on top of existing ED infrastructure.

    **Ground state computation:**
    1. Use construct_heisenberg_2d(4, 4, J=1.0, bc='obc') from ed_entanglement.py. This gives a 16-site system with 2^16 = 65536 dimensional Hilbert space. Use scipy.sparse.linalg.eigsh to get ground state.
    2. Verify ground state energy. For 4x4 OBC, the energy is not directly comparable to infinite-lattice e_0 = -0.6694 J/bond due to boundary effects, but should be in the right ballpark. As a sanity check, also compute PBC ground state and compare e_0/bond with literature.

    **Correlation functions:**
    3. Compute full correlation matrix C_{ij} = <S_i . S_j> for all 16x16 site pairs. Use S_i . S_j = S_i^z S_j^z + (1/2)(S_i^+ S_j^- + S_i^- S_j^+).
    4. Verify: C_{ii} = S(S+1) = 3/4 for all i (spin-1/2 identity).
    5. Staggered structure factor: S(pi,pi) = (1/N) sum_{i,j} (-1)^{r_i + r_j} <S_i . S_j> where (-1)^{r_i} = (-1)^{x_i + y_i} (checkerboard sublattice). Compute S(pi,pi)/N as proxy for m_s^2.
    6. Extract connected correlator as function of Manhattan distance d = |x_i - x_j| + |y_i - y_j| and Euclidean distance. Bin by distance and compute mean. On 4x4, cannot reliably fit power law, but can check sign pattern.

    **Sublattice structure in reduced states:**
    7. For a 2x1 subsystem (2 adjacent sites), compute rho_Lambda at positions where the left site is on sublattice A vs sublattice B. Compare trace distance ||rho_A - rho_B||_1. This tests whether the reduced state has sublattice-dependent structure (the key prediction for CORR-03).
    8. For OBC, also check corner vs edge vs bulk reduced states -- boundary effects will be present but bulk sublattice alternation should still be visible.

    **Red flag checks (from RESEARCH.md):**
    - If staggered structure factor ~ 0: likely symmetry sector issue. The singlet ground state has zero net magnetization but nonzero <S_i.S_j>(-1)^{i+j}. This is fine.
    - If all reduced states identical: the system is too symmetric. Check that OBC actually breaks lattice symmetry.

    Store all results in JSON: ground state energy, correlation matrix, staggered structure factor, sublattice rho comparison, reduced state trace distances.

    Include ASSERT_CONVENTION: SLD metric = 4 * Bures, J > 0 AFM, a = 1.
  </action>
  <verify>
    1. C_{ii} = 0.75 for all i (spin-1/2 identity S(S+1) = 3/4)
    2. sum_j C_{ij} = S_total(S_total+1)/N for singlet ground state (S_total = 0 gives sum_j <S_i.S_j> = 0 for each i)
    3. Staggered sign pattern: (-1)^{i+j} <S_i.S_j> > 0 for the majority of NN and NNN pairs
    4. S(pi,pi)/N > 0.05 (indicative of incipient Neel order at 4x4)
    5. Sublattice trace distance > 0.01 (rho_A != rho_B on OBC)
    6. Ground state energy is negative and extensive (E ~ -0.5 * N_bonds * J roughly)
    7. PBC sanity: ground state energy per bond closer to -0.6694 J
  </verify>
  <done>4x4 2D Heisenberg ground state computed, correlation matrix extracted, staggered structure factor S(pi,pi)/N shows Neel correlations, sublattice rho asymmetry detected on OBC</done>

<task type="auto">
  <name>Task 2: Fisher metric on 2D OBC lattice -- bulk non-vanishing test</name>
  <files>code/correlation_2d.py, data/correlation/corr_2d_4x4.json</files>
  <action>
    Compute the Fisher metric on the 2D OBC lattice to test the central CORR-03 prediction: g_bulk > 0 (non-vanishing in bulk, unlike 1D).

    **2D Fisher metric sweep:**
    1. Define the 1D parameterization for Fisher metric on 2D lattice: sweep a 2x1 (or 1x2) subsystem along one row, fixing y. For a 4x4 lattice with y=1 (interior row), sweep x = 0, 1, 2 (positions along the row).
    2. At each position x, compute rho_Lambda(x) via partial_trace of the ground state over all sites not in the 2-site subsystem at (x, y_fixed) and (x+1, y_fixed).
    3. Compute SLD Fisher metric g(x) = sum_{m,n} 2|<m|d_x rho|n>|^2 / (p_m + p_n) using the existing compute_qfim infrastructure from fisher_metric.py. The derivative d_x rho uses central finite difference: d_x rho = (rho(x+1) - rho(x-1)) / 2.
    4. Also sweep along a column (fix x, sweep y) for comparison.

    **Alternative: 2D plaquette subsystem.**
    5. For a richer test, use a 2x2 plaquette subsystem (4 sites). Sweep across the lattice. The plaquette reduced state captures 2D correlations better than a 1x2 strip. But 2x2 plaquette gives 2^4 = 16 dimensional reduced density matrix, which is fine computationally.
    6. For 2x2 plaquette on 4x4 OBC: positions (x,y) with x=0,1,2 and y=0,1,2 (9 positions). Compute rho_{2x2}(x,y). Fisher metric in x-direction: g_x(x,y) via central difference in x at fixed y.

    **Key comparison:**
    7. Load Phase 32 1D data from data/fisher/fisher_swap_1d.json. Compare g_bulk for:
       - 1D N=16 OBC: g_bulk ~ 0.22 * 16^{-2.75} ~ 0.22/4096^{0.69} (very small)
       - 2D 4x4 OBC: g at interior point (should be O(1) if Neel order helps)
    8. If g_bulk(2D) >> g_bulk(1D, N=16), this supports the Neel rescue hypothesis.
    9. If g_bulk(2D) ~ g_bulk(1D, N=16), the 4x4 may be too small to see the effect.

    **Bures cross-validation:**
    10. At one interior point, compute Bures metric independently: g_Bures = 2[1 - F(rho(x), rho(x+dx))] / dx^2 where F is the Uhlmann fidelity. Verify g_SLD = 4 * g_Bures in the infinitesimal limit (consistent with Phase 32 convention).

    Save Fisher metric profiles, comparison data, and Bures cross-check to JSON.
  </action>
  <verify>
    1. g(x) >= 0 everywhere (Fisher metric is positive-semidefinite by construction)
    2. g(x) > 0 at interior points where rho is full-rank (from FISH-02)
    3. g_SLD = 4 * g_Bures at test point (within 1e-4 relative error for dx=1)
    4. g_bulk(2D) compared with g_bulk(1D): report the ratio and whether it supports Neel rescue
    5. Boundary enhancement visible: g near edges > g in bulk (similar pattern to 1D)
    6. If 2x2 plaquette used: sublattice alternation visible in plaquette rho (A-centered vs B-centered plaquettes differ)
    7. All eigenvalues of rho are non-negative to machine precision
  </verify>
  <done>Fisher metric computed on 2D OBC lattice, interior g > 0 verified, comparison with 1D Phase 32 data completed, Bures cross-check passed. Result: g_bulk(2D) is [O(1)/O(1/N)/comparable to 1D] -- the specific value determines whether CORR-03 Neel rescue is numerically supported.</done>
</task>
