    **Structure of the argument:**

    **Step 1: State the problem precisely.**
    - In the gapped case (AKLT, easy-axis), Phase 32 FISH-01 gives smoothness via Hastings-Koma.
    - In the gapless Neel case (n=2, d>=2), Hastings-Koma does NOT apply. Correlations decay algebraically as 1/r^{d-1}.
    - Question: does the Fisher metric g_F(x) on reduced states rho_Lambda(x) remain smooth and non-vanishing in the bulk?

    **Step 2: Why 1D failed (recap from Phase 32).**
    - In 1D: Heisenberg chain is gapless, no LRO (Mermin-Wagner), ground state is a singlet with zero staggered magnetization.
    - Consequence: in the thermodynamic limit, the bulk reduced state rho_Lambda(x) becomes translation-invariant (independent of x). Therefore d_x rho -> 0 and g(x) -> 0.
    - Numerically: g_bulk ~ N^{-2.75} (Phase 32 Eq. 32.12).

    **Step 3: Why d>=2 is different -- the sublattice alternation mechanism.**
    - In d>=2 with Neel order: the ground state spontaneously breaks SU(2) to U(1), giving staggered magnetization m_s > 0 (m_s = 0.3074 for S=1/2 d=2 from QMC).
    - On a bipartite lattice, the sublattice magnetization alternates: sites on sublattice A have <S^z> ~ +m_s (in the symmetry-broken state), sites on sublattice B have <S^z> ~ -m_s.
    - For a subsystem Lambda of size |Lambda| >= 2 centered at position x: if x shifts by one lattice site, the subsystem moves from an A-centered position to a B-centered position. The reduced state changes because the sublattice character flips.
    - Formally: ||rho_Lambda(x+1) - rho_Lambda(x)||_1 = O(m_s) > 0 in the thermodynamic limit. This is the crucial difference from 1D where this quantity -> 0.
    - Therefore: d_x rho = O(m_s) and g_F(x) = O(m_s^2) > 0 in the bulk.

    **Step 4: Decompose rho into Neel + Goldstone contributions.**
    Write rho_Lambda(x) = rho_Neel(x) + delta_rho(x) where:
    - rho_Neel(x): the reduced state of the Neel mean-field state (product state with alternating magnetization). This has explicit sublattice dependence and gives g_F^{Neel}(x) = O(m_s^2).
    - delta_rho(x): correction from Goldstone mode (transverse spin-wave) fluctuations. In the NL sigma model:
      * The transverse correlator <n_perp(x) n_perp(0)> ~ 1/|x|^{d-1} (from Plan 01 CORR-01 result)
      * The correction to rho involves integrating over these correlations within the subsystem Lambda
      * delta_rho involves sum_{r in Lambda, r' outside Lambda} C_transverse(|r-r'|) which sums contributions ~ 1/r^{d-1}

    **Step 5: Bound the Goldstone correction to Fisher metric.**
    The Fisher metric receives contributions from both rho_Neel and delta_rho:
    - g_F(x) = g_F^{Neel}(x) + g_F^{Goldstone}(x) + cross terms
    - g_F^{Neel}(x) ~ m_s^2 (from sublattice alternation, O(1) and dominant)
    - g_F^{Goldstone}(x) involves d_x delta_rho, which depends on how the Goldstone corrections to rho change with x
    - The key integral: the Goldstone contribution to ||d_x rho||^2 involves sum ~ integral d^d r / r^{2(d-1)}
    - For d = 2: int dr / r = log(L/a) -- logarithmic divergence. The Goldstone correction to g_F grows as log(L) but the Neel contribution m_s^2 is O(1). At any finite L, g_F is well-defined and O(m_s^2) with log(L) corrections.
    - For d >= 3: int dr / r^{2(d-1)} * r^{d-1} = int dr / r^{d-1} converges for d >= 3 (since d-1 >= 2). The Goldstone correction is bounded. g_F is well-defined in the thermodynamic limit.

    **Step 6: State the conditional theorem.**
    Theorem (Fisher smoothness in Neel phase): Let H be the Heisenberg AFM on a d-dimensional bipartite lattice (d >= 2) with Neel long-range order (m_s > 0). Let rho_Lambda(x) be the reduced state of a subsystem Lambda of size |Lambda| >= 2 at position x, with OBC. Then:
    - (i) g_F(x) > 0 at interior points (where rho_Lambda is full-rank)
    - (ii) g_F(x) ~ C * m_s^2 + O(corrections) where corrections are bounded for d >= 3 and O(log L) for d = 2
    - (iii) For d >= 3: g_F has a well-defined thermodynamic limit g_F(x) -> g_F^{infty} > 0
    - (iv) For d = 2: g_F is well-defined at any finite L with possible logarithmic growth

    Hypotheses (state explicitly):
    - H1: Neel long-range order with m_s > 0 (proved for S >= 1 d >= 3 by DLS, QMC-established for S = 1/2 d = 2)
    - H2: Transverse correlations decay as 1/r^{d-1} (from spin-wave theory / NL sigma model, Plan 01)
    - H3: Subsystem is contiguous and |Lambda| >= 2 (need to see sublattice structure)
    - H4: Open boundary conditions (PBC gives g = 0 by translation invariance)

    **This argument is NOT rigorous.** It is a controlled physical argument based on the NL sigma model description. The conditions under which it holds are stated explicitly. The rigorous version would require proving that the NL sigma model correctly captures the reduced state structure, which is beyond current mathematical physics.
