    Continue the derivation document with the NL sigma model derivation for CORR-02.

    **Derivation of O(3) NL sigma model from Heisenberg AFM:**

    1. **Starting point:** H = J sum_{<ij>} S_i . S_j on d-dimensional bipartite lattice (J > 0 AFM). Ground state has Neel order for d>=2: <S_i> ~ (-1)^i m_s z_hat (breaking SU(2) to U(1)).

    2. **Sublattice decomposition:** Introduce staggered and uniform fields. On sublattice A ((-1)^i = +1): S_i = S[n(x_i) + l(x_i)/S]. On sublattice B ((-1)^i = -1): S_i = S[-n(x_i) + l(x_i)/S]. Here n is the staggered (Neel) field with n^2 = 1, and l is the uniform (canting) field with n.l = 0.

    3. **Coherent state path integral:** The lattice partition function Z = Tr[exp(-beta H)] is rewritten as a path integral over (n, l) using spin coherent states |Omega> parameterized by unit vectors on S^2.

    4. **Expand H in fields:** Nearest-neighbor coupling S_i . S_j for sites on different sublattices gives:
       - Leading: -S^2 z (nabla n)^2 a^2 (spatial gradient of staggered field)
       - Canting: -z S l^2 (potential for uniform fluctuations)
       where z is the coordination number (z=2d for hypercubic lattice, z=4 for square).

    5. **Integrate out l:** The l field is massive (gap ~ zJS). Gaussian integration produces a kinetic term (d_tau n)^2 for the staggered field.

    6. **Resulting action:**
       S_eff = (1/(2g)) int d^d x d tau [(d_tau n)^2 / c_s^2 + (nabla n)^2]

       with:
       - Coupling: g = d * c_s / (2 * rho_s) in d spatial dimensions
       - Or equivalently: 1/g = 2S/a^{d-1} in the classical (large-S) limit
       - Topological term: theta = 2*pi*S per unit cell. For d>=2 spatial dimensions, the topological term is absent (pi_2(S^2) topological sectors exist but the theta term structure is different from d=1).

    7. **Spin-wave velocity (CRITICAL -- must be SWAP-specific):**
       - Classical (LSWT): c_s^{cl} = 2*sqrt(d) * J * S * a = sqrt(2) * Ja for S=1/2, d=2 (square lattice)
       - Quantum renormalization: c_s = Z_c * c_s^{cl} where Z_c = 1.1765 +/- 0.0002 (Canali-Girvin-Wallin, PRB 45, 10131, 1992)
       - Quantum-renormalized: c_s = 1.1765 * sqrt(2) * Ja = 1.664 Ja (0.3% above QMC)
       - Independent verification: c_s = sqrt(rho_s / chi_perp) where rho_s = 0.180752(6) J, chi_perp = 0.065690(5) / J (Sandvik 2025). Gives c_s = sqrt(0.180752/0.065690) = 1.6588 Ja. Confirmed.
       - QMC benchmark: c_s = 1.65880(6) Ja (Sandvik 2025, arXiv:2601.20189)
       - This is the SWAP lattice spin-wave velocity since H_SWAP = H_Heis + const and the constant does not affect dynamics.

    8. **NL sigma model parameters (d=2, S=1/2 square lattice):**
       - g = 2 * c_s / (2 * rho_s) = c_s / rho_s = 1.6588 / 0.180752 = 9.178 (strong coupling!)
       - rho_s = 0.180752(6) J (spin stiffness)
       - chi_perp = 0.065690(5) / J (transverse susceptibility)
       - m_s = 0.307447(2) (staggered magnetization)
       - The fact that g > g_c (where g_c is the quantum critical coupling) means the square lattice is in the Neel-ordered (renormalized classical) regime.

    **Dimensional analysis (MANDATORY):**
    - [S_eff] = dimensionless (check: [1/g] = dimensionless, [int d^d x d tau] = [length^{d+1}], [(nabla n)^2] = [1/length^2], so total = [length^{d-1}] which requires... actually in d=2: [1/g] = dimensionless, [int d^2 x dtau] = [length^3], [(nabla n)^2] = [1/length^2], [(d_tau n)^2/c_s^2] = [time^2/(Ja)^2/time^2] = [1/(Ja)^2]... Need to be careful with units.

[(nabla n)^2] = 1. [int d^2 x dtau] = 1/J. [1/g] = dimensionless. So [S_eff] = 1/J, not dimensionless! There is a factor.
