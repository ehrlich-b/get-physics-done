  <name>Task 1: Derive the 3-step Lipschitz bound with explicit constants</name>
  <files>derivations/lipschitz-stability.tex</files>
  <action>
    Derive the Lipschitz bound |rho(P) - rho(P')| <= L * ||P - P'||_inf in three steps.
    Write the proof in derivations/lipschitz-stability.tex as a self-contained LaTeX document.

    **Setup.** State the theorem precisely:
    Let P, P' be irreducible row-stochastic matrices on Omega = B x M with |Omega| = n = |B| * |M|.
    Let pi, pi' be their stationary distributions. Let gap(P) = 1 - |lambda_2(P)| > 0.
    Let h_min > 0 be a lower bound on H_pi(B). Then:
      |rho(P) - rho(P')| <= L * ||P - P'||_inf
    where L is given explicitly below.

    **Step 1: Stationary distribution perturbation (cite Cho-Meyer 2001, Theorem 3.1).**
    State the bound: ||pi - pi'||_1 <= (1/gap(P)) * ||P - P'||_inf.
    Here ||P - P'||_inf = max_i sum_j |P_ij - P'_ij| is the row-sum norm (operator norm induced by vector inf-norm).
    Explain briefly: this follows from pi' - pi = pi'(P' - P)(I - P + Pi)^{-1} where Pi = 1*pi,
    with ||(I - P + Pi)^{-1}||_inf <= 1/gap(P).
    DO NOT re-derive this bound. Cite it as an established result.

    **Step 2: Mutual information continuity.**
    Let p = pi reshaped as joint distribution on B x M (pi_i -> p(b,m) where i = b*|M| + m).
    Similarly q = pi' reshaped. Then ||p - q||_1 = ||pi - pi'||_1.

    Bound |I_p(B;M) - I_q(B;M)| using I = H(B) + H(M) - H(B,M) and triangle inequality:
      |I_p - I_q| <= |H_p(B) - H_q(B)| + |H_p(M) - H_q(M)| + |H_p(B,M) - H_q(B,M)|

    For each term, apply the entropy continuity lemma (Csiszar-Korner 2011, Lemma 2.7; tight form: Audenaert 2007):
      For distributions on alphabet of size k with ||p - q||_1 = delta (our L1 convention):
      |H(p) - H(q)| <= (delta/2) * ln(k - 1) + h_bin(delta/2)
      where h_bin(x) = -x*ln(x) - (1-x)*ln(1-x) is binary entropy in nats.

    CRITICAL CONVENTION CHECK: The Fannes-Audenaert bound is stated with delta = ||p-q||_TV = (1/2)||p-q||_1 in many references. We use ||.||_1 throughout. If citing a reference that uses TV, convert: their delta_TV = our delta_L1 / 2. The bound above already uses our convention (delta/2 in the formula means we divide our L1 distance by 2 to get the TV distance used in the bound).

    Apply to each term with the appropriate alphabet size:
    - H(B): alphabet |B|, perturbation ||p_B - q_B||_1 <= ||p - q||_1 = delta (data processing inequality)
    - H(M): alphabet |M|, perturbation ||p_M - q_M||_1 <= delta
    - H(B,M): alphabet |Omega| = |B|*|M|, perturbation ||p - q||_1 = delta

    Combined MI bound:
      |I_p - I_q| <= (delta/2)[ln(|B|-1) + ln(|M|-1) + ln(|Omega|-1)] + 3*h_bin(delta/2)

    Define: C_I(n) = (1/2)[ln(|B|-1) + ln(|M|-1) + ln(|B|*|M|-1)] + 3*h_bin_coeff
    where h_bin_coeff captures the binary entropy contribution at small delta.

    For small delta: h_bin(delta/2) ~ (delta/2)*ln(2/delta), so the total is
      |I_p - I_q| <= delta * [(3/2)*ln(n) + O(ln(1/delta))]
    where n = |Omega|. For the Lipschitz constant, we need the worst-case coefficient at small delta,
    which is dominated by the (3/2)*ln(n) term.

    **Step 2b: H(B) continuity (needed separately for Step 3).**
    Also bound |H_p(B) - H_q(B)| <= (delta/2)*ln(|B|-1) + h_bin(delta/2).
    This is needed because rho involves the ratio I/H(B).

    **Step 3: Compose through rho = I*(1-I/H).**
    Write rho = f(I, H) where f(I, H) = I*(1 - I/H), I = I_pi(B;M), H = H_pi(B).

    Compute the Lipschitz constant of f:
      |f(I,H) - f(I',H')| = |I*(1-I/H) - I'*(1-I'/H')|

    Expand using the product rule for g(I,H) = I and h(I,H) = (1 - I/H):
      |f - f'| <= |I - I'| * |1 - I'/H'| + |I| * |(1-I/H) - (1-I'/H')|
      = |I - I'| * |1 - I'/H'| + |I| * |I/H - I'/H'|

    For the quotient I/H:
      |I/H - I'/H'| = |(I*H' - I'*H)/(H*H')|
      <= (|I - I'|*H' + I'*|H - H'|) / (H*H')
      <= |I - I'|/H + I'*|H - H'|/H^2    (using H' >= h_min and I' <= H')

    Since 0 <= I <= H and 0 <= 1-I/H <= 1:
      |f - f'| <= |I - I'| + I*(|I - I'|/H + I*|H - H'|/H^2)
      <= |I - I'| * (1 + I/H) + I^2/H^2 * |H - H'|
      <= 2*|I - I'| + |H - H'|    (using I/H <= 1, I^2/H^2 <= 1)

    Substituting Steps 1 and 2:
      |rho - rho'| <= 2*C_I*delta + C_H*delta = (2*C_I + C_H)*delta
    where delta = ||pi - pi'||_1 <= (1/gap(P)) * ||P - P'||_inf.

    **Final Lipschitz constant:**
      L = (2*C_I + C_H) / gap(P)
    where C_I = (1/2)[ln(|B|-1) + ln(|M|-1) + ln(|Omega|-1)] + 3*h_bin_max
    and C_H = (1/2)*ln(|B|-1) + h_bin_max
    with h_bin_max = max_{delta in (0,1)} h_bin(delta/2)/delta (a finite constant).

    For the leading behavior: L = O(ln(|Omega|) / gap(P)).

    More precisely, for small perturbations (delta << 1), the binary entropy terms are subleading, giving:
      L ~ [3*ln(|Omega|) + ln(|B|)] / (2*gap(P))

    State the domain of validity explicitly:
    - P irreducible with gap(P) > 0
    - P' irreducible (so pi' exists)
    - H_pi(B) >= h_min > 0 (B not degenerate)
    - The bound holds for any ||P - P'||_inf > 0 (not just small perturbations), but the constant C_I, C_H depend on the regime through h_bin

    Note: The factor 1/h_min does NOT appear in the final L because we used |I^2/H^2| <= 1 and |I/H| <= 1 which hold universally. The h_min dependence would appear only in a tighter analysis that tracks the actual I/H ratio. For the toy model h_min = ln(4) = 1.386, which is O(1), so this is not restrictive.

    Cite ref-lmc (Lopez-Ruiz, Mancini, Calbet 1995) when introducing rho as belonging to the LMC statistical complexity family. Cite ref-gell-mann-lloyd (Gell-Mann, Lloyd 1996/2003) for broader context of effective complexity measures.

  <name>Task 2: Verify limiting cases and consolidate the L formula</name>
  <files>derivations/lipschitz-stability.tex</files>
  <action>
    Add a "Limiting Cases and Remarks" section to derivations/lipschitz-stability.tex verifying the bound's behavior in all known limits.

    **Limiting case 1: Zero perturbation (P' = P).**
    When P' = P: ||P - P'||_inf = 0, so the bound gives |rho - rho'| <= 0, which is correct since rho(P) = rho(P).

    **Limiting case 2: Fast mixing (gap(P) -> 1).**
    L = O(ln(|Omega|) / gap(P)) -> O(ln(|Omega|)). This is finite and non-vacuous.
    The bound says: even for rapidly mixing chains, rho can change by at most O(ln(|Omega|)) per unit change in the kernel. This is reasonable because rho <= H(B)/4 <= ln(|B|)/4, so a Lipschitz constant of O(ln(|Omega|)) means the bound allows rho to change by its full range.

    **Limiting case 3: Slow mixing (gap(P) -> 0).**
    L -> infinity. This is correct: nearly reducible chains have stationary distributions that are extremely sensitive to small kernel perturbations. Physical interpretation: if the chain barely communicates between basins, tiny changes in transition probabilities can redirect all stationary mass.

    **Limiting case 4: Independent B, M (I(B;M) = 0).**
    At the boundary where I = 0, rho = 0. Near this boundary, rho ~ I (to leading order in I), so |rho - rho'| ~ |I - I'|. The bound correctly gives |rho - rho'| ~ C_I * ||P - P'||_inf / gap(P), which is consistent.

    **Limiting case 5: Perfect tracking (I(B;M) = H(B)).**
    At the boundary where I = H(B), rho = 0. Near this boundary, rho ~ (H - I), so |rho - rho'| ~ |H - I - (H' - I')|. The bound handles this via the 2|I-I'| + |H-H'| composition.

    **Limiting case 6: Toy model evaluation.**
    For the 7-chain toy model: |B| = |M| = 4, |Omega| = 16.
    Compute the explicit numerical value of L using the formula.
    - C_I = (1/2)[ln(3) + ln(3) + ln(15)] + 3*h_bin_max_term
    - C_H = (1/2)*ln(3) + h_bin_max_term
    - L = (2*C_I + C_H) / gap(P)
    State the gap(P) for the toy model's observer chain (this will be computed numerically in Plan 02, but state the formula here with gap as a parameter).

    **Remark on tightness:**
    Note that the 3-step composition uses worst-case bounds at each step. The actual Lipschitz constant of rho may be smaller because:
    (a) The entropy continuity bound is tight for worst-case distributions, not necessarily for the specific rho functional.
    (b) The product rule bound |2*|I-I'| + |H-H'|| may overcount because I and H are correlated (both depend on pi).
    Numerical comparison with L_numerical (Plan 02) will determine the tightness factor.

    **Remark on domain:**
    Restate the domain of validity: P must be irreducible with gap(P) > 0; P' must be irreducible; H_pi(B) > 0 (B not degenerate under pi). The Lipschitz bound holds on the open set of irreducible stochastic matrices.
