  <files>code/lipschitz_verification.py</files>
  <action>
    Create code/lipschitz_verification.py as a self-contained Python script (numpy only -- scipy is not available).

    **Reuse from composite_self_model.py:** Copy the entropy(), mutual_info(), rho(), stationary(), and chain construction functions from ~/repos/blog/research/experiential-measure/toy-model/composite_self_model.py. Preserve the exact numerical implementations. Do NOT import from that file (keep this script self-contained).

    **Implement the following functions:**

    1. `spectral_gap(P)`: Compute gap(P) = 1 - |lambda_2(P)| using numpy.linalg.eig. Return the absolute spectral gap.

    2. `random_stochastic_perturbation(P, epsilon)`: Generate a random perturbation P' of P with ||P - P'||_inf = epsilon.
       Method: For each row i, generate random perturbation e_i with sum(e_i) = 0 (to preserve row sums), scale to ||e_i||_1 = epsilon, add to P_i, clip to [0,1], renormalize row. Check that P' is a valid stochastic matrix (rows non-negative, sum to 1). Check irreducibility via spectral gap > 0. If P' is reducible, resample.

    3. `compute_rho(P)`: Compute rho(P) = I(B;M) * (1 - I(B;M)/H(B)) where pi = stationary(P), the joint distribution on B x M is pi reshaped to (|B|, |M|), I = mutual_info(joint), H = entropy(marginal_B).

    4. `L_proven(n_B, n_M, gap)`: Evaluate the proven Lipschitz constant from Plan 01's formula. Read the exact formula from the Plan 01 SUMMARY (02-01-SUMMARY.md).
       Expected form: L = (2*C_I + C_H) / gap where C_I and C_H are explicit functions of |B|, |M|.
       For the small-perturbation leading form: L ~ [3*ln(|B|*|M|) + ln(|B|)] / (2*gap).

    5. `L_numerical(P, n_samples, epsilon)`: Compute L_numerical = max over n_samples random perturbations of |rho(P) - rho(P')| / ||P - P'||_inf. Return both the max ratio and all individual ratios.

    **Build the 7-chain observer kernel** following composite_self_model.py chain #1 (the "observer" with alpha tracking and moderate noise). Use the SAME parameters as the original: N_B = N_M = 4.

    Include ASSERT_CONVENTION checks at the top of the script:
    - Verify entropy(np.array([0.25]*4)) == ln(4) (nats convention)
    - Verify mutual_info for independent B, M gives I ~ 0
    - Verify rho(I=0) = 0 and rho(I=H) = 0 boundary conditions

    Print all results clearly. The script should be runnable with `python3 code/lipschitz_verification.py` and produce all validation output.
  </action>
  <verify>
    1. ASSERT_CONVENTION checks pass: entropy in nats, MI = 0 for independent systems, rho boundaries correct.
    2. spectral_gap returns gap > 0 for the observer chain (irreducible).
    3. random_stochastic_perturbation produces valid stochastic matrices with correct ||P-P'||_inf.
    4. L_proven formula matches Plan 01 derivation (cross-check against 02-01-SUMMARY.md).
    5. For zero perturbation (epsilon = 0): |rho(P) - rho(P)| = 0.
    6. Code is numpy-only (no scipy dependency).
  </verify>
