    **Section 1: Setup and Definitions**

    **Section 2: Lemma (Faithful Embedding)**

    State and prove:
    Lemma: The observer faithfully represents V_{1/2} on a subspace of its Hilbert space H = C^n. Since V_{1/2} = R^16 and the T_a are linearly independent self-adjoint operators, faithful representation requires n >= 16. The standard inclusion iota: M_16(R) -> M_16(C) (viewing real matrices as complex matrices with zero imaginary part) embeds the T_a as self-adjoint elements of M_16(C) subset M_n(C).

    Key points of the proof:
    - The observer measures the physical degrees of freedom in V_{1/2} (these are the self-modeled observables)
    - Measurement requires representing T_a as self-adjoint elements of the observer's algebra
    - The T_a generate M_16(R) (256-dimensional), which embeds in M_16(C) via iota
    - This embedding preserves self-adjointness: T_a^dagger = T_a^T = T_a (real symmetric matrices are Hermitian)
    - The argument does NOT require n = 16 exactly; it requires n >= 16 for faithfulness. State the theorem for the restriction to the V_{1/2} sector.

    **Section 3: Proposition (Extended Sequential Product)**

    State and prove the Gudder-Greechie domain extension:
    Proposition: In any C*-algebra A, the Luders sequential product a & b = sqrt(a) b sqrt(a) extends naturally from the effect domain [0,I] to all self-adjoint elements with non-zero spectrum. Specifically:
    - For effects (spectrum in [0,1]): sqrt is the positive square root, always real. This is the Gudder-Greechie domain.
    - For general self-adjoint elements (spectrum may include negative values): sqrt is defined via the holomorphic (continuous) functional calculus of the ambient C*-algebra. For eigenvalue lambda < 0, sqrt(lambda) = i*sqrt(|lambda|) (principal branch).
    - The extended product is well-defined and associative when it exists.

    Explicitly state: T_a has eigenvalue -1/2, so T_a is NOT an effect. The Gudder-Greechie framework does not cover it. The observer's C*-structure provides the extension.

    **Section 4: Theorem (Observer-Induced Complexification) -- Route A**

    State the theorem:
    Theorem (Observer-Induced Complexification): Let V_{1/2} = R^16 carry the Cl(9,0) generators T_a with {T_a, T_b} = (1/2) delta_{ab} I_16, and let the observer's algebra be M_n(C)^sa (n >= 16) with the Luders sequential product (Paper 5 Thm 3.1). Then:
    (i) The observer's sequential product on the embedded T_a uses the holomorphic FC of M_n(C), yielding sqrt(T_a) = (1/sqrt(2))(P_+ + i P_-) where P_+/- are the spectral projections of T_a.
    (ii) For anticommuting T_a, T_b: sqrt(T_a) T_b sqrt(T_a) = (i/2) T_b.
    (iii) The sequential product exits M_16(R): Im(sqrt(T_a) T_b sqrt(T_a)) != 0 for all 72 anticommuting pairs.

    Proof (Route A -- Holomorphic FC):
    1. The observer is M_n(C)^sa. In a complex C*-algebra, the functional calculus for self-adjoint elements is the continuous FC from C(sigma(a)) -> A. For T_a with spectrum {+1/2, -1/2}, sqrt is computed via f: {+1/2, -1/2} -> C with f(lambda) = sqrt(lambda) using the principal branch.
    2. f(1/2) = 1/sqrt(2) and f(-1/2) = i/sqrt(2) (the unique value in C with f(lambda)^2 = lambda and Re(f(lambda)) >= 0). So sqrt(T_a) = (1/sqrt(2)) P_+ + (i/sqrt(2)) P_- = (1/sqrt(2))(P_+ + i P_-).
    3. This is the unique square root in M_n(C) satisfying the principal branch convention. There is no real-valued square root of T_a (that would require sqrt(-1/2) in R, which does not exist). The observer, being M_n(C)^sa, MUST use this complex FC. There is no alternative.
    4. Compute: sqrt(T_a) T_b sqrt(T_a) = (1/2)(P_+ + iP_-) T_b (P_+ + iP_-). Using {T_a, T_b} = 0 which gives P_+ T_b P_+ = 0 and P_- T_b P_- = 0, expand to get (1/2)(i P_+ T_b P_- + i P_- T_b P_+) = (i/2)(P_+ T_b P_- + P_- T_b P_+) = (i/2) T_b. Verify the last equality: T_b = P_+ T_b P_- + P_- T_b P_+ (since the diagonal blocks vanish). QED.
    5. Cite Phase 42 for numerical/symbolic verification of step 4 for all 72 pairs.
