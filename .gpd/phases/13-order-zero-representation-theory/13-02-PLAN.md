  <action>
    Create a comprehensive pytest test file that verifies the order zero condition and all related spectral triple properties at n=2.

    Implementation structure:

    1. Helper functions (parameterized by n):
       - build_swap_matrix(n): construct n^2 x n^2 SWAP matrix P with P_{(ij),(kl)} = delta_{il}*delta_{jk}
       - build_J_matrix(n): construct 2n^2 x 2n^2 matrix J_matrix = [[0, P], [P, 0]] (the linear part of J)
       - build_gamma_matrix(n): construct gamma = [[P, 0], [0, -P]]
       - build_pi_naive(a, n): construct pi(a) = block_diag(kron(a, I_n), kron(a, I_n))
       - build_pi_contragredient(a, n): construct pi(a) = block_diag(kron(a, I_n), kron(conj(a), I_n))
       - build_pi_o(b, J_mat, pi_func, n): compute pi_o(b) = J_mat @ conj(pi_func(conj(b).T, n)) @ J_mat
         CRITICAL: J is antilinear, so J A J^{-1} = J_matrix * conj(A) * J_matrix^{-1} = J_matrix * A-bar * J_matrix
         For pi_o(b) = J pi(b*) J^{-1}: first compute pi(b*) = pi(b^dagger), then apply antilinear conjugation:
         pi_o(b) = J_matrix @ conj(pi(b^dagger)) @ J_matrix
         Since conj(pi(b^dagger)) = conj(pi(b)^dagger) = pi(b)^T (transpose, NOT conjugate transpose),
         pi_o(b) = J_matrix @ pi(b)^T @ J_matrix

    2. Test: test_J_squared(n=2):
       - Verify J_matrix @ conj(J_matrix) = I (J^2 accounts for double conjugation)
       - Since J_matrix is real, J_matrix @ J_matrix = I

    3. Test: test_J_gamma_anticommute(n=2):
       - J is antilinear, so J gamma = J_matrix * conj(gamma) * ... but gamma is real
       - For real gamma: J_matrix @ gamma + gamma @ J_matrix should equal 0
       - Verify this is the zero matrix

    4. Test: test_gamma_pi_commute(n=2):
       - For each matrix unit E_ij (4 total at n=2):
         verify gamma @ pi(E_ij) - pi(E_ij) @ gamma = 0

    5. Test: test_order_zero_naive(n=2):
       - For each pair (E_ij, E_kl) with i,j,k,l in {0,1} (16 pairs):
         compute [pi_naive(E_ij), pi_o_naive(E_kl)] and verify it is the zero matrix
       - Use exact integer/rational arithmetic via SymPy matrices OR verify Frobenius norm < 1e-14 with NumPy

    6. Test: test_order_zero_contragredient(n=2):
       - Same as above but with contragredient action

    7. Test: test_pi_o_is_representation(n=2):
       - Verify pi_o(E_ij @ E_kl) = pi_o(E_ij) @ pi_o(E_kl) for all pairs
       - Verify pi_o(E_ij^dagger) = pi_o(E_ij)^dagger for all units
       - E_ij @ E_kl = delta_{jk} E_il (matrix unit multiplication)

    8. Test: test_pi_o_dimension(n=2):
       - Verify dim(span{pi_o(E_ij)}) = n^2 = 4 (the pi_o matrices are linearly independent)

    Use NumPy for the core computation (fast, exact for integer matrices). Use assertion with tolerance 1e-14 for floating-point, but prefer integer matrices where possible (matrix units are 0/1 matrices).

    IMPORTANT: J acts as J(v) = J_matrix @ conj(v). So:
    J A J^{-1} (v) = J(A(J^{-1}(v))) = J_matrix @ conj(A @ J_matrix @ conj(v))
                    = J_matrix @ conj(A) @ conj(J_matrix) @ v
                    = J_matrix @ A-bar @ J_matrix  (since J_matrix is real, conj(J_matrix) = J_matrix)

    For pi_o(b): pi_o(b) = J pi(b*) J^{-1} = J_matrix @ conj(pi(b*)) @ J_matrix = J_matrix @ pi(b)^T @ J_matrix
    (using conj(pi(b*)) = conj(pi(b)^dagger) = (pi(b)^dagger)^* = pi(b)^T for matrices with standard conjugation)
