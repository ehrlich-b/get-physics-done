  <name>Task 1: Derive pi_o(b) explicitly at general n</name>
  <files>derivations/07-order-zero-condition.md</files>
  <action>
    Compute pi_o(b) = J pi(b*) J^{-1} step-by-step for a general element b in M_n(C), on H = (C^n tensor C^n)_p + (C^n tensor C^n)_ap.

    CRITICAL: J is ANTILINEAR. Decompose J = C_matrix * K where K is complex conjugation. Then:
    pi_o(b) = J pi(b*) J^{-1} = C_matrix * conj(pi(b*)) * C_matrix^{-1} = C_matrix * pi(b)^T * C_matrix^{-1}
    (since conj(pi(b*)) = conj(pi(b)^dagger) = pi(b)^T).

    Step-by-step procedure:

    1. Write pi(a) on H in 2x2 block form (particle | antiparticle):
       NAIVE: pi(a) = diag(a tensor I_n, a tensor I_n)
       CONTRAGREDIENT: pi(a) = diag(a tensor I_n, a-bar tensor I_n)

    2. Write J as antilinear operator: J(psi, chi) = (PC chi-bar, PC psi-bar).
       In matrix form: J acts as J_matrix * conj(v) where J_matrix is a 2n^2 x 2n^2 permutation matrix encoding the sector-swap and SWAP.
       Specifically: J_matrix = [[0, P_matrix], [P_matrix, 0]] where P_matrix is the n^2 x n^2 SWAP matrix on C^n tensor C^n with entries P_{(ij),(kl)} = delta_{il} delta_{jk}.
       Verify: J^2 = J_matrix * conj(J_matrix) = J_matrix * J_matrix = I (since J_matrix is real and J_matrix^2 = I).

    3. For J^{-1}: since J^2 = 1, J^{-1} = J. So J^{-1}(psi, chi) = (PC chi-bar, PC psi-bar).

    4. Compute pi_o(b)(psi, chi) = J[pi(b*)(J^{-1}(psi, chi))]:
       a. Apply J^{-1}: (psi, chi) -> (PC chi-bar, PC psi-bar)
       b. Apply pi(b*): depends on which algebra action (naive or contragredient)
       c. Apply J to the result

    5. Track PC action explicitly:
       PC(v tensor w) = P(v-bar tensor w-bar) = w-bar tensor v-bar
       For an element sum_{ij} c_{ij} e_i tensor e_j:
       PC(sum c_{ij} e_i tensor e_j) = sum c_{ij}* e_j tensor e_i

    6. Derive pi_o(b) sector-by-sector. For the NAIVE action, compute:
       pi_o(b) acting on particle sector (psi):
       - J^{-1} maps (psi, 0) to (0, PC psi-bar)
       - pi(b*) acts on antiparticle sector: (b* tensor 1)(PC psi-bar)
       - J maps back: result lands in particle sector
       Track through to get pi_o(b)|_{particle}.

       pi_o(b) acting on antiparticle sector (chi):
       - J^{-1} maps (0, chi) to (PC chi-bar, 0)
       - pi(b*) acts on particle sector: (b* tensor 1)(PC chi-bar)
       - J maps back: result lands in antiparticle sector
       Track through to get pi_o(b)|_{antiparticle}.

    7. Simplify using PC(b* tensor 1)(PC)^{-1}:
       Key identity: if v tensor w is a tensor product, then
       PC(b* tensor 1)(PC)^{-1}(v tensor w) = PC(b* tensor 1)(w-bar tensor v-bar)
                                                = PC((b* w-bar) tensor v-bar)
                                                = v tensor (b* w-bar)-bar
                                                = v tensor (b^T w)
       This gives 1 tensor b^T -- the transpose action on the second factor.

    8. Conclude: pi_o(b) should act as 1 tensor b^T on each sector (for the naive action). Write the result in block form.

    9. If the naive action fails (pi_o doesn't have clean form), repeat with contragredient action.

    10. Document both results and identify which action is correct.

    Reference: van Suijlekom 2024, Ch. 3 for the standard computation pattern.
    Reference: Connes 1995 for the order zero condition definition.
    Reference: Paper 6 for J, P, gamma definitions.

    WARNING: Do NOT treat J as linear. The step conj(pi(b*)) = pi(b)^T is crucial and comes from the antilinearity.
    WARNING: Track which sector each vector lands in after each application of J.
  </action>

    Using the explicit pi_o(b) from Task 1, prove [pi(a), pi_o(b)] = 0 for ALL a, b in M_n(C).

    Proof strategy (assuming pi_o(b) = diag(1 tensor b^T, 1 tensor b^T) from Task 1):

    1. Write pi(a) in block form: pi(a) = diag(a tensor I, f(a) tensor I) where f(a) = a (naive) or a-bar (contragredient).

    2. Write pi_o(b) in block form: pi_o(b) = diag(1 tensor b^T, 1 tensor b^T) (or whatever form Task 1 gives).

    3. Compute the commutator block-by-block:
       [pi(a), pi_o(b)] = diag([a tensor I, 1 tensor b^T], [f(a) tensor I, 1 tensor b^T])

    4. For each block, use the tensor product commutator identity:
       [A tensor B, C tensor D] = AC tensor BD - CA tensor DB
       For A = a, B = I, C = I, D = b^T:
       [a tensor I, I tensor b^T] = (a*I) tensor (I*b^T) - (I*a) tensor (b^T*I) = a tensor b^T - a tensor b^T = 0

       This is the KEY STEP: operators on different tensor factors commute.

    5. Verify this works for BOTH sectors. For the antiparticle sector with contragredient action:
       [a-bar tensor I, 1 tensor b^T] = a-bar tensor b^T - a-bar tensor b^T = 0
       (same argument, different first factor).

    6. Conclude: [pi(a), pi_o(b)] = 0 for all a, b in M_n(C), at general n.

    7. As a completeness check, verify on the basis: [pi(E_{ij}), pi_o(E_{kl})] = 0 for all matrix units. By bilinearity, this extends to all a, b.

    8. Document the proof clearly, noting:
       - Which algebra action was used (naive, contragredient, or both)
       - The essential mathematical fact (operators on different tensor factors commute)
       - The role of J in converting the algebra action to a transpose action on the complementary factor
       - That the sector-swap in J is handled by the block-diagonal structure

    If order zero FAILS for the naive action:
    - Characterize the failure mode: which pairs (a,b) have nonzero commutator?
    - Try the contragredient action
    - If both fail, document the obstruction and trigger the stop/rethink condition

    References: Connes 1995 for the condition definition; van Suijlekom Ch. 3 for the standard argument.
