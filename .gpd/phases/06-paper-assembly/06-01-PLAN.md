    Write the full content for Sections 1-3 of paper/main.tex. This is the hardest expository content.

    **Section 1: Introduction (~2 pages)**

    Opening paragraph: State the result -- a finite-dimensional system that faithfully models itself is necessarily governed by complex quantum mechanics. One operational premise, no assumptions about the complex field, Hilbert space, or C*-algebraic structure.

    Second paragraph: Context -- the quantum reconstruction program. Brief history (Hardy 2001, CDP 2011, Dakic-Brukner, Masanes-Mueller). The common thread: multiple independent axioms, often assuming local tomography or complex structure. This work's contribution: derive rather than assume.

    Third paragraph: The main theorem, stated informally. "If a system's state space admits a faithful self-model -- an isomorphic copy through which the system can probe and update its own state -- then the state space is necessarily M_n(C)^sa for some n, equipped with the Luders sequential product and conjugate-transpose involution."

    Fourth paragraph: Comparison table (LaTeX tabular). Columns: Program | Year | Premises | Complex selection. Rows: Hardy (5, Simplicity), CDP (6, Purification), Dakic-Brukner (4, LT assumed), Masanes-Mueller (3, tomographic locality assumed), Barnum-Mueller-Ududec (4, energy observability), This work (1, derived from faithful tracking). Add a footnote: "We count 'premises' as independent operational postulates. Our single premise requires four standing structural assumptions (see Assumptions 1-4)."

    Fifth paragraph: Outline of the paper with forward references to sections.

    Sixth paragraph: Honest statement of scope. "Our result holds for finite-dimensional systems (Assumption 1), assumes a faithful self-model exists (Assumption 2), uses the minimal composite construction (Assumption 3), and restricts to simple EJAs (Assumption 4). We discuss the status of each assumption in Section 7."

    **Section 2: Preliminaries (~1.5 pages)**

    2.1 Order Unit Spaces: Definition, effects [0,1]_V, sharp effects (projections), spectral decomposition. Cite Alfsen-Shultz. State Assumption 1 (finite-dimensional spectral OUS).

    2.2 Compressions and Peirce Decomposition: P-projections C_p, Peirce 0/1/2 spaces, the key identity: C_p + C_{p^perp} = pinching map (NOT identity) in non-commutative OUS. Cite Alfsen-Shultz.

    2.3 Sequential Products: Definition from vdW (Def. 2). List S1-S7 with names. State vdW Theorem 1 (S1-S7 => EJA). State vdW Theorem 3 (SP + LT => C*-algebra). These are BLACK BOX theorems -- cited, not proved.

    2.4 Self-Modeling: Define self-modeling as an order-isomorphism phi: V_B -> V_M between a system B and its model M, plus the operational triple "prepare-update-measure." State Assumption 2 (faithful self-model: phi is an isomorphism). This is the SOLE operational premise.

    **Section 3: The Self-Modeling Sequential Product (~3 pages)**

    This is the main novel contribution. Write it as a self-contained construction.

    3.1 From Self-Modeling to Sequential Product: The operational logic: "test effect a on B, update M via phi, test effect b on B" yields a composite measurement described by a . b. Define the product for sharp effects: p . q = C_p(q) (compression product). Verify it maps effects to effects.

    3.2 The Naive Extension and Its Failure: Attempt to extend from sharp to general effects via spectral decomposition: a = sum lambda_i p_i => a . b = sum lambda_i C_{p_i}(b). Show this FAILS S3 (unitality) because sum C_{p_i} = pinching map, not identity. The Peirce 1-space is annihilated. This is NOT a bug -- it reveals that the compression sum misses off-diagonal information.

    3.3 The Corrected Product via Peirce Feedback: The key insight: the self-modeling feedback loop provides the missing Peirce 1-space contribution. Write the general formula:
      a . b = sum_i lambda_i C_{p_i}(b) + sum_{i<j} f(lambda_i, lambda_j) P_{ij}(b)
    Derive the positivity bound: |f(lambda_i, lambda_j)| <= sqrt(lambda_i lambda_j) from the requirement 0 <= a . b <= 1.

    3.4 Faithful Self-Modeling Selects the Maximum: Argument that faithful phi (isomorphism, maximum information preservation) selects f = sqrt(lambda_i lambda_j), saturating the positivity bound. State this as a proposition with careful language (selection principle, not uniqueness theorem). Write the corrected product:
      a . b = sum_i lambda_i C_{p_i}(b) + sum_{i<j} sqrt(lambda_i lambda_j) P_{ij}(b)
    Note: on M_2(C)^sa, this coincides with the Luders product sqrt(a) b sqrt(a). The equivalence is a CONSEQUENCE, not a premise.

    3.5 Circularity Check: Brief paragraph addressing the sqrt question. "The scalar function sqrt(lambda_i lambda_j) operates on real eigenvalues -- it is arithmetic, not C*-functional calculus. No operator square root, trace, inner product, density matrix, or Hilbert space structure is used in the construction." Reference the full circularity audit.

    - Citations to vdW with explicit theorem numbers
