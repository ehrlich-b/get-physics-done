      xi^mu = (x^1, c_s^2 t, 0, ..., 0)     ... (35.4)

    Step 10: Derive theta = 0 at the bifurcation surface. The expansion is:

      theta = nabla_mu xi^mu     ... (35.5)

    At B (where xi^mu = 0 and d_mu xi^nu = const from the Killing equation), compute:

      theta|_B = nabla_mu xi^mu|_{x^0=0, x^1=0}

    For the boost Killing vector: nabla_0 xi^0 = d(x^1)/dx^0 = 0 (partial derivative at fixed x^0 = 0) and nabla_1 xi^1 = d(c_s^2 t)/dx^1 = 0 (at t = 0). So theta|_B = 0.

    More precisely: for any Killing vector, nabla_mu xi^mu = 0 everywhere (Killing's equation nabla_(mu xi_nu) = 0 implies tracelessness). So theta = 0 is automatic for a Killing vector field, not just at the bifurcation surface.

    Step 11: Derive sigma = 0 at the bifurcation surface. The shear is:

      sigma_{mu nu} = nabla_{(mu} xi_{nu)} - (1/(d-1)) theta h_{mu nu}     ... (35.6)

    - theta = 0 (no expansion) -- from Killing's equation
    - sigma = 0 (no shear) -- from Killing's equation
