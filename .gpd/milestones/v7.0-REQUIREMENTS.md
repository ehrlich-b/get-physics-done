# Requirements: Arrow of Time, Complexification, and Evolutionary Selection

**Defined:** 2026-03-24
**Core Research Question:** Is complexification forced by thermodynamic necessity -- self-modelers requiring free energy, entropy gradients, time-orientation, chirality -- closing Gap C via rho selection?

## Primary Requirements

### Derivations

- [ ] **ENTR-01**: Prove or disprove monotonic entropy increase for reduced states under iterated Luders sequential products on the SWAP lattice (address non-unital issue explicitly)
- [ ] **ENTR-02**: If entropy can decrease, identify precise conditions (initial state class, interaction strength) under which it increases
- [ ] **CHIR-01**: Verify that Weyl (chiral) spinor bundles require time-orientation and space-orientation (Lawson-Michelsohn reference)
- [ ] **CHIR-02**: State three-consequence theorem: single choice u in S^6 determines gauge group + chirality + time-orientation
- [ ] **LAND-01**: Prove self-modeling cycle (test body, update model, test again) necessarily processes information and costs free energy via Landauer's principle
- [ ] **LAND-02**: Derive quantitative bound: mutual information I(B;M) maintained by self-model requires at least kT * I(B;M) of free energy per update cycle
- [ ] **LAND-03**: Address coherence loophole: can self-modeling operate using quantum coherence without classical free energy?
- [ ] **GRAD-01**: Derive entropy gradient theorem via at least one route (Direct, Boundary, or rho selection)
- [ ] **GRAD-02**: State Gap C resolution as theorem: complexification is a selection effect (rho = 0 for non-complexified observers)

### Calculations

- [ ] **CALC-01**: Compute entropy change under one Luders sequential product step for SWAP dynamics on M_2(C) x M_2(C) (explicit 2-qubit case)
- [ ] **CALC-02**: If Phase 4 succeeds, compute minimum initial entropy consistent with self-modeling and compare to Penrose's estimate S_initial ~ 10^88

### Validations

- [ ] **VALD-01**: Verify the chirality-time link against Lawson-Michelsohn explicitly (not just citation)
- [ ] **VALD-02**: Cross-check Landauer bound with Sagawa-Ueda generalized Jarzynski framework
- [ ] **VALD-03**: Verify rho = 0 for equilibrium self-modelers (I(B;M) = 0 at thermal equilibrium)

## Follow-up Requirements

### Extended Analysis

- **EXTD-01**: Relationship between CP violation and entropy gradient rate
- **EXTD-02**: Full rho profile over cosmological time (rho peaks at I = H/2)
- **EXTD-03**: Evolutionary game theory formalization (ESS for complexified self-modelers)
