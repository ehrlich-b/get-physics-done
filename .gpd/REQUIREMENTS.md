# Requirements: Gap C Algebraic Closure via C*-Measurement Maps

**Defined:** 2026-03-29
**Core Research Question:** Does a C*-observer's Peirce multiplication maps on V_{1/2} = O^2 (inside h_3(O)) force complexification, closing Gap C?

## Primary Requirements

### Algebraic Verification

- [ ] **ALGV-01**: Compute V_1 Peirce multiplication L_{E_{11}} on V_{1/2} = O^2 explicitly; confirm V_1 = R*E_{11} bottleneck (L_a = (alpha/2)*Id on V_{1/2})
- [ ] **ALGV-02**: Compute V_0 Peirce multiplication maps L_b: V_{1/2} -> V_{1/2} for all b in V_0 = h_2(O); search for J with J^2 = -Id in the 10-parameter operator family on R^16
- [ ] **ALGV-03**: Characterize the observable algebra on V_{1/2} generated by all Peirce multiplication compositions (products, commutators, quadratic representations); determine whether it contains an element J with J^2 = -Id

### Representation Theory

- [ ] **REPR-01**: Construct explicit Cl(9) representation on V_{1/2} = R^16; verify Spin(9) action; identify J_u (the would-be 10th Clifford generator from Krasnov's G_SM = Stab_{Spin(9)}(J_u) reformulation)
- [ ] **REPR-02**: Determine whether J_u lies in the observable algebra computed in ALGV-03; this is the decisive test for algebraic closure of Gap C

### Impossibility and Selection

- [ ] **IMPL-01**: If algebraic forcing fails (ALGV-02, ALGV-03, REPR-02 all negative): prove impossibility theorem (no Peirce-generated operator gives J^2 = -Id) and formalize the selection/bootstrap argument as Gap C's honest status

### Integration

- [ ] **INTG-01**: State Gap C resolution as one of: (a) algebraic theorem (if REPR-02 positive), (b) selection principle with precise self-consistency conditions (if IMPL-01), or (c) precisely characterized remaining gap; update Paper 7 accordingly

## Follow-up Requirements

### Extended Analysis

- **EXTD-01**: Analyze quadratic representation U_a(x) = 2a*(a*x) - a^2*x for richer structure transmission beyond linear Peirce multiplication

- **EXTD-03**: Formalize EGT backup route via constructor theory impossibility (no complexification -> no chirality -> no chemistry -> no self-modelers)
